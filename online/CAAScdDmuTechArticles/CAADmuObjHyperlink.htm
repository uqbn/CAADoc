Hyperkink Object

 

See Also UseCases Properties Methods

 




Hyperlink object represents hyperlink object. Hyperlinks are usefull when a user wants to add URLs to a component.

The following image shows an example of hyperlinks added to a product:

Using the Hyperlink object, one can add automatically URLs.

Hyperlinks are managed thru a Hyperlinks collection which can be retrieved, on a product, using the GetTechnologicalObject method with the "Hyperlinks" parameter value.
