Marker3D Object

 

See Also UseCases Properties Methods

 




Marker3D object represents 3D markers. 3D markers are usefull when a user wants to add some textual information on a component.

The following image shows an example of 3D markers describing some components:

Using the Marker3D object, one can add automatically textual information.

Marker3Ds are managed thru a Marker3Ds collection which can be retrieved, on a product, using the GetTechnologicalObject method with the "Marker3Ds" parameter value.
