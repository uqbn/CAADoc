The GetTechnologicalObject method of the Product object, with the "Mechanisms" parameter value for argument, retrieves, if it exists, the collection of Mechanism objects associated with the product. It also retrieves with the "Dessups" parameter value for argument, the collection of Dressup objects associated with the product.

The GetWorkbench method applied to a ProductDocument object with the "KinematicsWorkbench" parameter value allows to retrieve a KinematicsWorkbench object. From this object, you can retrieve the Mechanisms collection.
