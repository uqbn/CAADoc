Inertia Object

 

See Also UseCases Properties Methods

 
 

An inertia is used when it is needed to compute inertia data on a product. The Inertia object associated to a product is retrieved using the GetTechnologicalObject method on the corresponding Product object with the "Inertia" parameter value for argument. 

he computed data are :

  • mass
  • center of gravity
  • inertia matrix
  • principal axes
  • principal moments
