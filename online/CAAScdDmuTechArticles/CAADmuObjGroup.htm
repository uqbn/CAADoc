Group Object

 

See Also UseCases Properties Methods

 




Group objects represent groups. Groups are needed when a set of components has to be processed as a whole for a given task.

The following image shows an example of group which gather all the red components :

Using the Group object, one can automatically build a group by adding or removing components.

Groups are managed thru a Groups collection that can be retrieved, on a Product object using the GetTechnologicalObject method with the "Groups" parameter value.
