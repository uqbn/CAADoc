AnnotatedView Object

 

See Also UseCases Properties Methods

 
 

An AnnotatedView is the object that represents an annotation. An annotation is used when the user needs to exchange remarks made during the review of a product.

The following image shows an example of annotations:

The annotated view gathers a persistent form of a 3D viewpoint and, eventually, a comment, a sound and a series of marker2D objects.

Information about the viewpoint can be retrieved from the annotated view:

The comment can be read or edited.

The sound file can be added, changed or removed.

The Marker2D objects attached to the annotated view are accessed thru a Marker2Ds collection.

Annotated views are managed thru an AnnotatedViews collection.
