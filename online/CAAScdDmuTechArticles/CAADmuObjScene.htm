Scene Object

 

See Also UseCases Properties Methods

 

A scene is created when the user needs to explore an alternative design for a product and to present this alternative design to colleagues.

The following image shows an example of scene with moved components:

Using the Scene object, it is possible to automatically prepare this alternative.

Scenes are managed thru a Scenes collection. The Scenes collection is aggregated by a SceneWorkbench object that can be retrieved from the ProductDocument object using the GetWorkbench method with the "SceneWorkbench" parameter value.
