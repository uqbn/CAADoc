The GetItem method of the Part and Product objects allows you to retrieve, if it exists, the aggregated BehaviorExtension object. Call it as follows:

Set BehExtension = Root.GetItem("CATGetBehaviorExtensions")

Where Root is either the Part or Product root object.

Select the behavior associated with this BehaviorExtension object using its method SelectBehavior. The selected Behavior object can itself aggregate a collection of Behavior objects .
