Installing Rich Application Presentation Designer
Technical Article

Abstract

The purpose of this article is to explain how to install IDD - Rich Application Presentation Designer. Be sure that you have the latest version.

Installing Rich Application Presentation Designer

  1. Double click the IDDSetup.msi, the installation wizard for the IDD - Rich Application Presentation Designer Product.

    This file is located in the <ENOVIA_Studio_INSTALLATION_PATH>/<OS>/resources/MSI folder, where <ENOVIA_Studio_INSTALLATION_PATH> is the pathname of the folder into which the ENOVIA Studio CD-ROM was unloaded and <OS> is the name of the operating system folder as explained in [1].

    The wizard displays:

  2. Click Next. The Select Installation folder displays.

  3. Select the path to install Rich Application Presentation Designer. Click Next.

  4. Click Next. The installation is now complete.

  5. Once the installation is complete, you must update your installation of Visual Studio, running devenv.exe with the option /setup:

    "C:\Program Files\Microsoft Visual Studio 8\Common7\IDE\devenv.exe" /setup if Visual Studio is installed in this directory.

Removing Rich Application Presentation Designer

  1. Click Add or Remove Programs in Windows Control Panel.
  2. Select the IDD - Rich Application Presentation Designer item from the list and click Remove.

    The tool is now removed from Windows.
