Installing CAA V5 API Products in Batch Mode with UNIX
The start batch command lets you install CAA V5 without the graphical user interface. You can install all CAA Version 5 APIs configurations and products in batch mode.

Note that, if your Version 5 software is delivered on more than one CD-ROM, you must copy all the software to the same directory.
To perform a batch installation, change directory to the CD-ROM mount point and enter the command:
./start

with at least one of the arguments below.

start [-code "RTInstallDir"] [-u "CAAInstallDir"] [-newdir] -list|-all|-l "list_to_unload" [-s] [-v] [-h]

start Command Syntax

-code "RTInstallDir" Specifies the directory where you have installed the run time part of the configurations and products for which you want to install the APIs. Make sure the directory path is enclosed like this: "install dir" if the directory name contains spaces. The default installation directory is:
/usr/DassaultSystemes/B22
-u "CAAInstallDir" Specifies the CAA APIs installation directory, if you wish to install the CAA APIs in another directory than the one used to install the run time part of the configurations and products. Make sure the directory path is enclosed like this: "unload dir" if the directory name contains spaces. The default unload directory is:
/usr/DassaultSystemes/B22
-newdir Creates the unload directory if it doesn't exist
-list Lists the configurations and products on the CD-ROM
-all Unloads all the configurations and the products on the CD-ROM
-l "list_to_unload" Specifies the list of configurations and/or products to unload. You have to type the list of configurations and/or products, which you can obtain by running the command using the "list" argument. In the list, configuration names look like this: ME2.slt, and product names look like this: KIN.prd. These are the names you must type with a caa2 suffix, such as ME2.slt.caa2 or KIN.prd.caa2. Separate the names using a blank.
The arguments -list, -all and -l "list_to_unload" are mutually exclusive.
-s Silent mode
-v Verbose mode
-h Displays help
Reminder: If you run the command without arguments, the installation will be started using the graphical user interface.

Examples

The following examples install CAA Version 5 APIs in batch mode.

  • To install the CAA Version 5 APIs of the Conf configuration and of the Prod product in the same directory than the run time in verbose mode, type the following command: 
    start -code "RTInstallDir" -l "Conf.slt.caa2 Prod.prd.caa2" -v

    where:

    RTInstallDir The directory where you have installed the run time part of the Conf configuration and the Prod product
    Conf.slt.caa2 The CAA V5 APIs of the Conf configuration
    Prod.prd.caa2 The CAA V5 APIs of the Prod product

     

  • To install the CAA Version 5 APIs of the Conf configuration and the Prod product in another directory than the run time in verbose mode, type the following command: 
    start -code "RTInstallDir" -u "CAAInstallDir" -newdir -l "Conf.slt.caa2 Prod.prd.caa2" -v

    where:

    RTInstallDir The directory where you have installed the run time part of the Conf configuration and the Prod product
    CAAInstallDir The directory where you want to install the CAA V5 APIs of the Conf configuration and the Prod product
    Conf.slt.caa2 The CAA V5 APIs of the Conf configuration
    Prod.prd.caa2 The CAA V5 APIs of the Prod product
