Installing CAA RADE V5 Products in Batch Mode with Windows
The StartB command lets you install CAA RADE without the graphical user interface. You can install all CAA Version 5 RADE products in batch mode.
To perform a batch installation:
  1. Select Start->Programs->Command Prompt to open a Command Prompt window.
  2. Go to the drive containing the CD-ROM, then to the Intel folder.
  3. Enter the command: 
    StartB

    with the appropriate arguments.

StartB [-lic LicenseDir/License.lic] [-u CAARADEInstallDir] [-newdir] [-ident] [-D] [-noLang] [-noFonts] [-TCKmode NoServer|Client|Server] [-TCKServerName name] [-TCKServerPort port] [-noCheckTCKPort] [-TCKBoot] -list|-all|-l "list_to_unload" [-v] [-h]

StartB Command Syntax

-lic "LicenseDir/License.lic" Enrolls a nodelock license. Specifies the directory and the file containing the license certificate to enroll. Make sure the directory path is enclosed like this: "license dir" if the directory name contains spaces.
-u "CAARADEInstallDir" Specifies the CAA RADE installation directory. Make sure the directory path is enclosed like this: "unload dir" if the directory name contains spaces. The default unload directory is:
C:\Program Files\Dassault Systemes\B22
-newdir Creates the unload directory if it doesn't exist
-ident Identifies the installation compared with another one of the same level. The identifier must be a string of uppercase characters [A-Z] and digits [0-9] without blanks and must not exceed 20 characters.

The identifier for your new installation is preceded by an underscore and is visible:

  • In the environment name
  • In the appropriate registry entries
  • In the Start->Programs->CAA RADE->Tools menu
  • In the Add/Remove Programs control.
-D Specifies the environment directory. The default environment directory is:
C:\Documents and Settings\All Users\Application Data\DassaultSystemes\CATEnv
-noLang Specifies the user interface languages to exclude. By default, all available languages are installed. The English language cannot be excluded. In addition to English, the available languages, associated symbols to use in this option, and associated language file installation directory are:
Language Symbol Directory
French fr French
German ge German
Italian it Italian
Japanese jp Japanese
Korean kr Korean
Simplified Chinese ch Simplified_Chinese

To exclude a given language, use the associated symbol. For example, to exclude the French language, enter:

-noLang "fr"

To exclude all the languages, excepted english, enter:

-noLang all

Note that the choice you made a first installation is definitive. You cannot add or remove languages later when installing additional configurations and/or products.

The language file installation directory is installed in the installation directory specified by the -u option, in os\resources\msgcatalog, where os is

Symbol Operating System
intel_a Windows 32-bit
win_b64 Window 64-bit
-noFonts Specifies that language-indexed fonts must not be installed. By default, language-indexed fonts are installed. These fonts are useful when installing the japanese, korean, and simplified chinese languages only.

Note that the choice you made a first installation is definitive. You cannot add or remove fonts later when installing additional configurations and/or products.

The language-indexed fonts are installed in the installation directory specified by the -u option, in os\resources\fonts\Stroke. os has the same values than above. The following files are installed

Language File
Japanese KANJ.font
Korean KOHG.font
Simplified Chinese SICH.font
Traditional Chinese TRCH.font
-TCKMode NoServer|Client|Server Specifies the tools access mode. Default is NoServer
-TCKServerName serverName Specifies the TCK server name. Useless with NoServer mode.
-TCKServerPort serverPort Specifies the TCK server port. Useless with NoServer mode.
-noCheckTCKPort Does not check if the communication port specified using -TCKServerPort is free.
-TCKBoot Boots the TCK server at restart. This option is of no use if CAA RADE is not installed on a server.
-list Lists the configurations and products on the CD-ROM
The arguments -list, -all and -l "list_to_unload" are mutually exclusive.
-all Unloads all the configurations and the products on the CD-ROM
The arguments -list, -all and -l "list_to_unload" are mutually exclusive.
-l "list_to_unload" Specifies the list of configurations and/or products to unload. You have to type the list of configurations and/or products, which you can obtain by running the command using the "list" argument. In the list, configuration names look like this: CDC.slt, and product names look like this: DMC.prd. Separate the names using a blank.
The arguments -list, -all and -l "list_to_unload" are mutually exclusive.
-v Verbose mode
-h Displays help

Examples

The following examples install CAA RADE in batch mode.

  • To install
    • In the e:\RADE\V5R22 directory to be created by the command and the environment in e:\RADE\CATEnv
    • For the english and german languages and without language-indexed fonts
    • The CAA RADE CDC configuration and the DMC product in verbose mode

    Type the following command:

    StartB -u "e:\RADE\V5R22" -newdir -D "e:\RADE\CATEnv" -noLang "fr it jp ko ch" -noFonts -l "CDC.slt DMC.prd" -v
  • To install in the existing e:\RADE\V5R22 directory the JDV configuration, type: 
    StartB -u "e:\RADE\V5R22" -l "JDV.slt"
