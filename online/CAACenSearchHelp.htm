CAA Encyclopedia Full-Text Search Help

Performing Simple, Fuzzy Searches

The default settings let you perform a basic, fuzzy search throughout the entire online CAA Encyclopedia.

For example, if you are looking for information about the web, simply enter the string web in the Search for field and click the Go button. The search engine searches for the string web, and displays the list of articles that include this string, either in a word or at the beginning of the word, in the Found in list. For example, the articles that include web are found, as well as those that include Webtop, or WebInfo, the search being not case-sensitive.

For each found article, this list displays on a single line:

  1. The name of the domain to which the article belongs. This is usually a modeler
  2. The article title.

These lines are sorted in alphabetical order. In our example, the search for the string web lists all the documents that contain the string:

3d com Navigator - Creating a Driver
3d com Navigator - Developing a Driver
3d com Navigator - Driver Overview
3d com Navigator - Portal Applications
...

Double-click an item in the Found in list to open the document containing the searched string.

The page containing the searched string is then displayed in the bottom frame of the Search window, so your search query is kept. A Find Next push button helps you find the searched string throughout the article.

You can switch from Scope to Results by clicking the Scope button. Scope lets you define the search scope as defined below in Full Search and Restricted Search. Results shows the list of articles matching the search criteria. You can check Open in a new window to open the found articles in a new window. Otherwise, they are opened in the bottom frame of the Search window.

As soon as you have double clicked an article, a Find Next push button appears, and the window jumps to the first searched string occurrence that is highlighted. You can navigate from one occurrence to the other by successively pressing Find Next. If you open the article in a new window, a combo is also displayed. It is of no use if you searched for a single word, but will be useful for multi-word searches.

Full Search and Restricted Search

You can search the entire CAA Encyclopedia by selecting All documents in the Search in field. This is the default. You can also restrict your search to a given solution, set of solutions, modeler, or set of modelers, by appropriately selecting the searched domain in the Search in field.

Searching Solutions

  1. Uncheck the All documents box
  2. Check the Solutions box if it is not already checked
  3. Select the solutions you want to search in by successively clicking their names
  4. Type in the searched string in the Search for box
  5. Click Go

The string is searched for in the selected solutions only. Click Reset to deselect the selected solutions.

Searching Modelers

  1. Uncheck the All documents box
  2. Check the Modelers box if it is not already checked
  3. Select the modelers you want to search in by successively clicking their names
  4. Type in the searched string in the Search for box
  5. Click Go

The string is searched for in the selected solutions only. Click Reset to deselect the selected modelers.

Searching for Exact Words

Check the Match whole word option, enter the exact string you are searching for, and then click the Go button.

This time, if you search for the string web, the search engine will find only the articles that contain web, and not those that contain Webtop or WebInfo.

Searching for Exact Phrases

In the Search mode pull-down list, select the Exact phrase option.

Enter the exact phrase you want to search for. For example, enter the phrase What You Will Learn With This Use Case, and then click the Go button. Only those documents containing exactly the phrase you searched for will be listed.

Searching Using AND and OR Operators

You can use operators to search for several words or strings not forming a phrase (AND), or one of a list of words or strings you specify (OR).

In the Search mode pull-down list, select the AND option. Specify the words you want to search for, and then click the Go button.

For example, search for documents containing both the strings toolbar and icon. To search for either toolbar or icon, select OR in the Search mode pull-down list, enter the words toolbar and icon, then click the Go button. A combo is displayed in front of the Find Next button to let you select the string to find when clicking Find Next. In case of an OR search, a pop-up may appear if one of the searched string was not found.

Search Limitations

The following special characters are ignored: / \ - &

The search is not case-sensitive.

The search does not apply to the C++ API, Automation API, and Java API sections of the CAA V5 Encyclopedia.
