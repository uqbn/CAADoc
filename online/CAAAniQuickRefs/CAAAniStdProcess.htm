The standard processes for CATAnalysisCharacCollector manipulation
Quick Reference

Abstract

This article explains the list of processes provided with the analysis modeler in order to convert the CATAnalysisCharacCollector data. It includes the following section:

  • SPMProcessAbsoluteValue
  • SPMProcessAbsOrderEigenValue
  • SPMProcessComplex
  • SPMProcessComponent
  • SPMProcessInterval
  • SPMProcessLamina
  • SPMProcessLinear
  • SPMProcessNormalize
  • SPMProcessPrincipalValues
  • SPMProcessSmi

We have defined some standard processes. They are used to filter values or to apply a mathematical computation on the input characteristic. All these processes will be listed below. 

Process Name

Process Definition

 Input Parameters
SPMProcessAbsoluteValue This process is used to compute the absolute value of a scalar. No parameter
SPMProcessAbsOrderEigenValue This process is used to reorder an eigen tensor (depending on the absolute value). No parameter
SPMProcessComplex This process filters the given part of complex ( real , imaginary, phase or modulus). One INTEGER parameter : real =0, imaginary=1, phase=2, modulus=3 (default value : real)
SPMProcessComponent This process filters the given component. One INTEGER parameter : the component number to keep (0=C1, 1=C2,...) (default value : 0=C1)
SPMProcessInterval This process filters the values of characteristic belonging to intervals defined by their limits (up to 5 coefficients). The defined intervals may be included or excluded.

  I-----------I-----------I-----------I-----------I-----------I-----------I

-00     Coef1   Coef2    Coef3   Coef4    Coef5     +00

 From one to six INTEGER parameters : describing if intervals are included ( 1=[coefi,coefj], 2=[coefi,coefj[, 3=]coefi,coefj], 4=]coefi,coefj[ ) or excluded (0).

From zero to five FLOAT parameters giving the limits of intervals (Coefi values).
SPMProcessLamina This process filters the given lamina number One INTEGER parameter : the lamina number to keep. (default = first lamina =0)
SPMProcessLinear This process computes y=ax+b  (x is the input characteristic, y the output characteristic, a and b are given coefficients). Two FLOAT parameters : a and b coefficients ( default values : a=1.f b=0.f)
SPMProcessNormalize This process is used to normalize a vector No parameter
SPMProcessPrincipalValues This process is used to change a full or symmetrical tensor to an eigen tensor No parameter
SPMProcessSmi This process filters the given layer (upper, mid, lower or upper_lower) One INTEGER parameter : describing the layer to keep. (upper = 0 , mid = 1, lower = 2) (default  value = upper = 0). The upper_lower option needs two INTEGERS (upper = 0 and lower = 2)
