Meshing Connectivity

The standard connectivity's provided with the meshing modeler
Quick Reference

Abstract

The following reference article shows the ways some connectivity's are provided by default by the meshing model. It defines the way faces, edges and nodes of the elements are defined. The identifier BAR, BAR3 ... are the one you should use for programming.

  • Linear Bar.
  • Parabolic Bar.
  • Linear Triangle.
  • Parabolic Triangle.
  • Linear Quadrangle.
  • Parabolic Quadrangle.
  • Linear Tetrahedron.
  • Parabolic Tetrahedron.
  • Linear Pyramid.
  • Parabolic Pyramid.
  • Linear Wedge.
  • Parabolic Wedge.
  • Linear Hexahedron.
  • Parabolic Hexahedron.
  • Multiple Node Element: SPIDER.
  • Multiple Node Element: SNAKE.
