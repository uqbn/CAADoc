JavaReplay

Finds a directory or a file in a workspace
Quick Reference

Name

JavaReplay Replays the applet or the java application given in arguments of this command

Synopsis

JavaReplay MyApplet arg1 arg2

[Top]

Description

Runs the applet or the java application. Using this command, the ODT could be launched with every Java RunTime Environment just setting the ADL_ODT_JRE environment variable.

[Top]

Examples

To launch the applet MyApplet, use this command line:
JavaReplay MyApplet

[Top]

Environment Variables

WhichDir uses the following environment variables:

ADL_ODT_JRE Contains the Replay Java RunTime Environment

[Top]

Diagnostics

Possible exit status values are (the JavaReplay command retunrs the exit status value returned by java or jview):

0 Successful completion
>0 Failure because any of the following
