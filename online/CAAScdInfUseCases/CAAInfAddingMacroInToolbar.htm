Adding a Macro as a Command in a Toolbar

You can easily customize your toolbars by adding icons to trigger your favorite macros. This reduces the macro access time for the macros you often run.

This task explains how to add a macro as a command in a toolbar.
  1. Select the Tools->Macro->Macros... command to display the Macro dialog box, and select the macro you want to add to make sure that the directory path of the file containing the requested macro is concatenated with those recognized by CATIA. Then click Cancel
 
  1. Select Tools->Customize , then the Commands tab, and Macros. Drag the macro name to the toolbar you wish. The default icon
    2. is used
 
  1. To select another icon than the default one, click Show Properties, and browse the currently used icons by clicking
    2. . Select the appropriate icon and drag the macro name.
 
  1. You can also use the icon browser by clicking

    2. .
