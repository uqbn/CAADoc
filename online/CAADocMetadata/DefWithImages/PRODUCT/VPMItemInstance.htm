V521_GRAPHIC_PROP   
 RoleIt enables user to save graphic properties (color and transparency) and Hide/Show information for the Part Instances and exploded Product instances into ENOVIA V5 database and consequently retrieve the same data from ENOVIA V5 to CATIA V5.
The value is as follows: H_S;R,G,B;T
  • H_S: Hide/Show information. It is an integer which can only be equal to 0=Hide or 1=Show.
  • R,G,B: Color information. It is represented by three integers separated by a comma. Each integer value is in the range 0-250.
  • T: Transparency information. It is an integer which value is in the range 0-255.
