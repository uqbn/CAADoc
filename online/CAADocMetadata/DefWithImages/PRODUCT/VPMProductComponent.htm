VPMProductComponent Class Characteristics

name VPMProductComponent
type Class
Role The product component allows to give a functional view to an assembly. A ProductComponent tree can be defined for one ProductRootClass. One or several ItemInstances of the ProductRootClass can be linked to a same ProductComponent.
