Data Type ConfigExpression

Kind: complexType (defining a structure)

Namespace: urn:com:dassault_systemes:ENOVWSProdStructIntf

Role: Configuration Expression is used to create modifications and configuration handlers. Specifications and Milestones cannot be created using the configuration expression, the existing specification and modifications will be applied to the newly created configuration handler or modification. Configuration expression will create only one configuration handler or Modification per call. Date and range can be created using the configuration expression.
