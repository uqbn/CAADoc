Data Type SessionToken

Kind: complexType (defining a structure)

Namespace: urn:com:dassault_systemes:CATJWSInfra

Role: Token to represent a user session in a ENOVIA LCA V5. SessionTokens can be obtained by invoking the the bindToApplication method of the ENOPosApplicationBinderImpl web service. They are also needed to close sessions using the releaseFromApplication method of the ENOPosApplicationBinderImpl web service.
