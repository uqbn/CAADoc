Data Type ENOVIA

Kind: complexType (defining a structure)

Namespace: urn:com:dassault_systemes:plmdataset:ENOVIA

Role: General purpose data structure to exchange data with the ENOVIA LCA V5 server. This data structure can contain the most common objects stored inside a ENOVIA LCA V5 server: product classes, products, part instances, actions, etc. The exact content of the structure depends on the web service method being invoked.
