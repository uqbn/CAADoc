adl_ren_cs

Renaming change set
Quick Reference

Abstract

The adl_ren_cs command renames a change set.

  • Synopsis
  • Usage
  • Options
  • Examples
  • References

Synopsis

adl_ren_cs change_set new_change_set_name [-h|-help|-?]

Usage

This command renames the change set <change_set> to <new_change_set_name>.

Options

adl_ren_cs accepts the following options:
 

-h|-help|-? Help of the command
change_set Name of the change set or identifier prefixed by "csid:".
new_change_set_name New name of the change set to rename.

[Top]

Examples

Renaming a change set.

>adl_ren_cs feb_RootT1Feb_6 Bug12Fix
Renaming the change set "feb_RootT1Feb_6" to "Bug12Fix".
