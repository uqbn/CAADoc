adl_rm_cs

Deleting a change set
Quick Reference

Abstract

The adl_rm_cs command deletes a change set.

  • Synopsis
  • Usage
  • Options
  • Examples
  • References

Synopsis

adl_rm_cs change_set  [-h | -help | -?]

Usage

This command removes the specified change set.
The command fails if the change set has included (created or moved in) changes.

[Top]

Options

adl_rm_cs accepts the following options:

-h|-help|-? Help of the command
change_set Name of the change set or identifier prefixed by "csid:".

Examples

Deletion of the change set "MyChangeSet": 

adl_rm_cs MyChangeSet
