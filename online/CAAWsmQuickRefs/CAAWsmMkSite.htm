adl_mk_site

Creating a site
Quick Reference

Abstract

The adl_mk_site command creates a new site. 

  • Synopsis
  • Usage
  • Options
  • Examples
  • References

Synopsis

adl_mk_site site_name  [-h | -help | -?]

Usage

To see all the existing site's names, run the [1] adl_ls_site command. To rename an existing site, run the [2] adl_ren_site command.

[Top]

Options

adl_mk_site accepts the following options:

-h|-help|-? Help of the command
site_name Name of the site to create

[Top]

Examples

Creation of the "MyNewSite" site:

adl_mk_site MyNewSIte
