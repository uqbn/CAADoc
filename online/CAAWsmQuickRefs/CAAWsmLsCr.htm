adl_ls_cr

Listing Incident Request
Quick Reference

Abstract

The adl_ls_cr command displays the list of the RIs in a given workspace. It is an internal Dassault-Systemes command. 

  • Synopsis
  • Usage
  • Options
  • Examples
  • References

Synopsis

adl_ls_cr [workspace_name] [-h | -help | -?]

Usage

This command displays, the list of the Incident Request (RI) for a given workspace.

The list of RIs is displayed with the following information:

  • NUMRI: number of the RI
  • CODRI: code associated with the RI
  • WORKSPACE: name of the workspace associated to the RI
  • BASE: name of the database in which the workspace is located

Those RIs have been associated with the workspace, when the [1] adl_promote command has been run.

[Top]

Options

adl_ls_cr accepts the following options:

-h|-help|-? Help of the command
workspace_name Name of the workspace to examine. Default is current one..

Examples


Display all the RIs associated with the workspace "MyWs": 

adl_ch_ws MyWs
adl_ls_cr
or
adl_ls_cr MyWs

 

Display all the RIs of the "MyWs" workspace. In this case, no RIs exist:  

adl_ls_cr NUMRI CODRI WORKSPACE BASE ------- ----- ------------------------------ ------------------------------ 0 record(s) selected.

 

Display all the RIs of the "MyWorkspace" workspace:  

adl_ls_cr MyWs NUMRI CODRI WORKSPACE BASE ------- ----- ------------------------------ ------------------------------ 0267343 A MyWorkspace MyDataBase 0267495 A MyWorkspace MyDataBase 2 record(s) selected.
