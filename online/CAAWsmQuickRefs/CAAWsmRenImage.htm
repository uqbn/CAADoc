adl_ren_image

Renaming an image
Quick Reference

Abstract

The adl_ren_image command renames an image for an existing workspace in an existing tree.

  • Synopsis
  • Usage
  • Options
  • Examples
  • References

Synopsis

adl_ren_image image_name new_image_name [-ws workspace_name] [-h | -help | -?]

Usage

This command renames an image and changes the symbolic name of an existing image. To rename the projection directory run the [1] adl_set_image_dir command.

The change of name is done giving the old name followed by the new name.

To create a new image, run the [2] adl_mk_image command.

[Top]

Options

adl_ren_image accepts the following options:
 

-h|-help|-? Help of the command
image_name Name of the image
new_image_name New name of the image
-ws workspace_name Name of the workspace the image to rename is associated with.

Examples

Rename the "MyWsImage" image in the "MyNewImageName" of the "MyWs" workspace: 

adl_ch_ws MyWs
adl_ren_image MyWsImage MyNewImageName
or
adl_ren_image MyWsImage MyNewImageName -ws MyWs
