adl_ls_site

Listing SCM site(s)
Quick Reference

Abstract

The adl_ls_site command displays the name of the current SCM site.

  • Synopsis
  • Usage
  • Options
  • Examples
  • References

Synopsis

adl_ls_site [-program [-sep | -separator char]] [-out file_path]  [-h | -help | -?]

Usage

The format of this command is like any adl_ls_... command but this one will always display only one line: it corresponds to the name of the current SCM site.

The name of the current site is registered when creating this site. To create a new site, run the [1] adl_mk_site command.

The "-program" option displays information in a specific output mode. For more details, see the article explaining this option.

[Top]

Options

adl_ls_site accepts the following options:
 

-h|-help|-? Help of the command
-program Display for programs: the identifier and the name of the current site are displayed
-sep char Separator between each field for programs (space " " by default)
-separator char Separator between each field for programs (space " " by default)
-out file_path Path of the file where the results are stored.

[Top]

Examples

Display the site's names:

adl_ls_site

 

Display the site's names in a program mode, with the character "|" used as separator:  

adl_ls_site -program -sep "|"
