adl_ren_tree

Renaming a tree
Quick Reference

Abstract

The adl_ren_tree command renames a tree managed on the current site.

  • Synopsis
  • Usage
  • Options
  • Examples
  • References

Synopsis

adl_ren_tree ws_tree_name  new_ws_tree_name  [-h | -help | -?]

Usage

This command renames a tree existing on a given site. To know all the existing trees on a given site, run the [2] adl_ls_tree command. To create a new tree, run the [1] adl_mk_tree command. To update characteristics of a tree, run the [3] adl_set_tree command.

[Top]

Options

adl_ren_tree accepts the following options:

-h|-help|-? Help of the command
ws_tree_name Name of the tree to be renamed
new_ws_tree_name New name of the workspace tree

Examples

Rename the "MyTree" tree in "NewTreeName" tree: 

adl_ren_tree MyTree NewTreeName
