adl_ren_ws

Renaming workspaces
Quick Reference

Abstract

The adl_ren_ws command renames a workspace. . 

  • Synopsis
  • Usage
  • Options
  • Examples
  • References

Synopsis

adl_ren_ws workspace_name new_workspace_name [-h|-help|-?]

Usage

This command renames a workspace.

Once the workspace is renamed, you have to run the [1] adl_ch_ws command to be in this new renamed workspace.

If you want to change the path to a workspace image, use the adl_set_image_dir command instead.

[Top]

Options

adl_ren_ws accepts the following options:

-h|-help|-? Help of the command
workspace_name Name of the workspace to be renamed
new_workspace_name New name of the workspace

[Top]

Examples

Rename the workspace "MyWs" in "MyNewWs":   

adl_ren_ws MyWs MyNewWs

 

Rename the workspace "MyWs" in "MyNewWs"? then run the adl_ds_ws command:

adl_ch_ws MyWs
adl_ren_ws MyWs MyNewWs
