User Command Reference

All SCM users commands by alphabetic order
Quick Reference

Abstract

Here are all the SCM users commands by alphabetic order. 
adl_attach To attach a framework or a module
adl_attach_is To attach a framework in a workspace for an inter-site transfer
adl_ch_resp To change the responsible of a component (framework, module) in a workspace tree
adl_ch_cs Select the current change set in the current workspace or create a new change set in the current workspace
adl_ch_ws Select the current workspace and set working environment
adl_chk To check that all files are managed 
adl_ci To check-in all checked-out files in an image
adl_close_cs To close a change set
adl_co To officially check-out a file
adl_collect To collect current promotion requests in current workspace
adl_detach To detach a framework or a module
adl_detach_is To detach a framework in a workspace for an inter-site transfer
adl_diff_ws_rev To display differences between workspace's revisions
adl_ds_cs To display change set's information
adl_ds_chg To display change information on a given object in the current workspace image
adl_ds_promo To display modifications of the last promotion request
adl_ds_ws To display workspace's information
adl_ds_ws_is To display workspace's information regarding intersite transfers
adl_forget_chg To cancel last changes on a software object
adl_get_file_content To extract the file contents corresponding to a given identifier
adl_gui To launch the federate graphic tool
adl_import To import changes from one workspace into another one
adl_import_ws To import changes of a whole workspace
adl_link_ws To declare an existing workspace in another workspace tree
adl_lock_req To lock promotion requests to a workspace
adl_ls_cfg_br To list the configuration branch identifiers associated with a workspace
adl_ls_co To list all official and private checked-out files
adl_ls_cr To list change request on current workspace (internal DS)
adl_ls_cs To list change sets associated to local changes in the current workspace
adl_ls_db To list existing SCM databases of current site
adl_ls_fw To list attached and not attached frameworks or modules in a workspace
adl_ls_fw_is To list attached and not attached frameworks in a remote workspace for an inter-site transfer
adl_ls_merge To list all merges still to be solved in a workspace
adl_ls_out To list associations between user files/directories and database objects
adl_ls_priv To list all private checked-out files
adl_ls_promo To list promotion requests (current and past) from a workspace
adl_ls_req To list current promotion requests to a workspace
adl_ls_resp To list framework or module responsible
adl_ls_transfer To list all transfers associated to the current workspace
adl_ls_tree To list all workspace trees registered on current site
adl_ls_type To list the predefined file types/suffixes
adl_ls_ws To list all workspaces in a one or many workspace trees
adl_ls_ws_rev To list all workspace's revisions
adl_mirror_is To transfer all content in an inter-site transfer
adl_mk_data To create a data module
adl_mk_dir To create/register a new directory
adl_mk_elem To register a new file
adl_mk_fw To create a framework
adl_mk_image To create an image for an existing workspace
adl_mk_mod To create a code module
adl_mk_priv To privately check-out a file
adl_mk_ws To create a workspace in a workspace tree
adl_mv To move/rename an object
adl_photo To freeze the current state of a workspace
adl_project To extract an old version of an object
adl_promote To deliver modifications from the current workspace
adl_promote_is To deliver modifications from the current workspace in an inter-site transfer
adl_publish To publish the current state of a workspace
adl_refresh To refresh an image against the associated workspace
adl_ren_cs To rename a change set
adl_ren_image To rename an image
adl_ren_ws To rename a workspace
adl_reorg_cs To reorganize the modifications of a change set
adl_restore_image To restore/repair an image
adl_rm To remove a framework, module or object
adl_rm_cs To remove a change set
adl_rm_image To remove an image
adl_rm_priv To privately uncheck-out a file
adl_rm_promo To remove last promotion from current workspace
adl_rm_req To remove a promotion request
adl_rm_ws To remove a workspace from all trees
adl_search_cs To search all the change sets created in a workspace
adl_set_cs To set/change characteristics of a change set
adl_set_image To set/change characteristics of an image
adl_set_image_dir To change the path of an image
adl_set_merge To set workspace's behavior for solving merges
adl_set_ws To set/change characteristics of a workspace
adl_site_transfer To exchange changes on shared frameworks between SCM sites
adl_solve_merge To solve all merges in a workspace
adl_solve_merge_is To solve all merges in a workspace in an inter-site transfer
adl_sync To get modifications in current workspace from parent workspace
adl_sync_is To get modifications in local current workspace from remote parent workspace in an inter-site transfer
adl_unco To officially uncheck-out a file
adl_unlink_ws To remove a workspace from a workspace tree
adl_unlock_req To unlock promotion request to a workspace
adl_unrm To resuscitate a removed framework, module or object
