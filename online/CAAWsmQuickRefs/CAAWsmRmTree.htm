adl_rm_tree

Deleting a tree
Quick Reference

Abstract

The adl_rm_tree command deletes a workspace tree.

  • Synopsis
  • Usage
  • Options
  • Examples
  • References

Synopsis

adl_rm_tree ws_tree_name  [-h | -help | -?]

Usage

This command deletes a workspace tree created with the [2] adl_mk_tree command. You must give the name of the tree to be deleted.
A tree name is unique on a given site. To know the name of the trees already created, run the [1] adl_ls_tree command.

[Top]

Options

adl_rm_tree accepts the following options:

-h|-help|-? Help of the command
ws_tree_name Name of the tree to be deleted

Examples

Suppression of the tree "MyTree" : 

adl_rm_tree MyTree
