adl_ren_site

Renaming a site
Quick Reference

Abstract

The adl_ren_site command renames a site. 

  • Synopsis
  • Usage
  • Options
  • Examples
  • References

Synopsis

adl_ren_site new_site_name  [-h | -help | -?]

Usage

To see all the existing site's names, run the [1] adl_ls_site command.

[Top]

Options

adl_ren_site accepts the following options: 

-h|-help|-? Help of the command
new_site_name Name of the new site

[Top]

Examples

Update of the current site into "MyRenamedSite" site's name:

adl_ren_site MyRenamedSIte

[Top]

References

[1] adl_ls_site: Listing all existing sites
