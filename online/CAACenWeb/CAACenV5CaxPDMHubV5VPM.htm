Cax/Document and PDM Object Hub

PDM Object Hub pages deal with data import-export scenarios. ENOVIA VPM V4 addresses import-export scenarios between V5 data and VPM V4. ENOVIA V5 addresses import-export scenarios between ENOVIA V5 and an external PDM system. And finally, File xPDM deals with scenarios enabling file base V5 documents to be exported to an external PDM system.
