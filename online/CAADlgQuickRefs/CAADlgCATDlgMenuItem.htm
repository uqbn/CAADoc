User Interface - Dialogs

CATDlgMenuItem

Base class for menu items
Quick Reference 
CATDialog
  |
  +---CATDlgMenuItem

CATDlgMenuItem is the base class for all dialog menus. Its derived classes are:

  • CATDlgPushItem
  • CATDlgCheckItem
  • CATDlgRadioItem
  • CATDlgSeparatorItem.
Inheritance
The parent of a instance of a class that derives from CATDlgMenuItem must be an instance of the CATDlgSubMenu or CATDlgContextualMenu classes.

[Top]

Programmer's Guide

A CATDlgMenuItem can be constructed as the base class of one of its derived class only.
