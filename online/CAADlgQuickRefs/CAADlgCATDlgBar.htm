User Interface - Dialogs

CATDlgBar

Base class for bars
Quick Reference 
CATDialog
  |
  +---CATDlgBar

CATDlgBar is the base class for status bars and toolbars. Its derived classes are:

  • CATDlgStatusBar
  • CATDlgToolBar.
Containment Parent
The parent of a instance of a class that derives from CATDlgBar must be an instance of a class that derives from the CATDlgDocument class.

Programmer's Guide

A CATDlgBar instance can be constructed as the base class of one of its derived class only.
