"Dummy" physical types

The IMPORTED physical types can be used for import scenario. By default the features that are created for this import can not know the exact physical types that will be created. The  IMPORTED entity can be created for each kind of set. This physical type is recommended as default physical type for the features that manage the import.

Physical type Category Usage
IMPORTED_MATERIAL MATERIAL Used to manage import of materials
IMPORTED_AXIS AXIS Used to manage import of local axes
IMPORTED_PROPERTY PROPERTY Used to manage import of physical properties.
IMPORTED_RESTRAINT RESTRAINT Used to manage import of boundary conditions
IMPORTED_LOAD LOAD Used to manage import of external conditions
IMPORTED_MASS MASS Used to manage import of non structural masses
