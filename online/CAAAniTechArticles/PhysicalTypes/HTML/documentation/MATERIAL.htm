"Material"  physical types

The Material Set is dedicated to receive material data inside the field model. It includes explicit entity objects with physical types compatible with MATERIALS. The explicit entities for representing materials are:

  • MATERIAL_ISOTROPIC defines an isotropic material that can handle structural and thermal behavior with the following characteristics:

The SHEAR_MODULUS represents the elastic Yield stress. This characteristic have to be compared with the VON Mises Stress of the elements. The Hardening parameters are used for plasticity behavior: it defines a linear hardening for stress values from SHEAR_MODULUS to YIELD_STRENGTH_X
