Tool Configuration Key Commands

A reference of TCK commands
Quick Reference

Abstract

This article provides a quick access to the TCK commands, sorted using the following categories:

  • Environment Setup
  • User Commands
  • Administrator Commands

Any TCK command displays its usage when setting either -h, -help or -? option.

For instance, execute tck_list -h to get a description of the supported options.

Environment Setup
To set up the current environment to access TCK commands tck_init
User Commands
To set up the current environment to access the tools associated with a given TCK tck_profile
To list all TCKs defined on local host or at site level tck_list
To register a local installation or to refresh its information tck_update_local
Administrator Commands
To launch a TCK server on current host tck_server
To test or to stop a TCK server from any host tck_server_admin
To create a new TCK along with associated tools tck_create
To modify information about a given TCK tck_modify
To remove a TCK or to remove a tool from a TCK tck_remove
