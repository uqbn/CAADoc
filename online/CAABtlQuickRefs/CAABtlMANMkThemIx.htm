Summary of Commands Around mkmk

Links to the mkmk family commands
Quick Reference

Mkmk being the main Code Builder tool, there exist others tools to manage either the prerequisites between frameworks or the frameworks themselves.

These command are provided with mkmk so they are available as soon as your path has been updated to access mkmk.

The commands to describe the prerequisites links are:

  • mkCopyPreq to copy the prerequisite frameworks into a workspace
  • mkGetPreq to construct a concatenation of prerequisites
  • mkPrintPreq to display the direct and indirect prerequisite frameworks

The commands to manage the frameworks are:

  • mkCopyFw to copy frameworks
  • mkRmFw to delete frameworks into a workspace
  • mkRemoveDo to delete derived objets

The commands to prepare a runtime environment are:

  • mkCreateRuntimeView to create a run time view
  • mkrun to execute a CAA program.
