With this popup you can display old command output.

The list of launched commands is displayed. Only commands of the day are available, others are destroyed, and their outputs can not be display anymore. You can display the output corresponding to a selected command or remove it.

Output can be displayed in a standard text editor, either in the CAA output page. With the CAA Output, you can use usual fonctionnalities (ex: double-click to show compilation error in source file).
