With this popup you can specify the way you want to access to prerequisites

There are 3 modes to access prerequisites. In all cases, you must indicate the root directory(ies) of workspace(s) where they can be found.

  • copy prerequisites locally in your workspace
  • set a reference to others workspaces
  • copy prerequisites in a local directory, different from your workspace directory. You must also specify the directory where prereqs have to be copied.
    Note: This mode executes a mkCopyPreq to copy Prereq frameworks needed for buildtime, to the given local directory, Then the mkGetPreq command is launched in order to prepare the runtime environment. So after wizard execution, buildtime and runtime environments are ready for use.
