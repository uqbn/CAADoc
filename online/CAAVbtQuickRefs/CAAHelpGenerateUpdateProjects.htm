With this popup you can select the framework(s) and module(s) for which a project must be generated.

The folders which appear in the left side of the window correspond to frameworks in and can be expanded/collapsed to respectively display or hide internal modules. Double-click on any framework/module to select it for project generation (it then appears on the right side of the window). You can also select one framework/module by a simple click and then click on the "right arrow" button. A framework/module which appears on the right side can be unselected by clicking on it and then clicking on the "left arrow" button.

 

Note:

  • If you select a framework, projects will be generated for it and for all the modules it contains.
  • Conversely if you select a framework and some modules from this framework, a project will be generated for this framework and the selected modules, other modules won't be processed.
