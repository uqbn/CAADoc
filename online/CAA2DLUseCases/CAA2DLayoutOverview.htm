2D Layout for 3D Design Modeler Overview

The 2D Layout for 3D Design is a new generation product that lets you design 3D models in an advanced 2D drafting-like production environment, enabling you to create layout views based on 2D geometry, while getting the most of other key capabilities such as dress-up, dimensions, annotations, 2D components, use-edges, constraints, and outputs of profiles and planes, for example. Once your design is laid out in 2D sheets and views, you will be able to print it directly or to generate a drawing sheet from it, to add views that will illustrate it.
 

A Layout is a collection of sheets, along with information on the drafting standard (ISO, ANSI, ASME and JIS).

The 2D Layout Sheet corresponds to the paper space, where the following elements are positioned:

  1. One 2D Layout main view: this view supports the geometries directly created in the sheet
  2. One 2D Layout background view: this view is dedicated to frame,  title block, revision blocks and bill of materials
  3. Any number of interactive 2D Layout Views.

The 2D Layout view may contain:

  • 2D geometry: Geometry created by using the 2D Layout for 3D Design interactive commands. The CAA Sketcher modeler is used to create 2D geometry.
  • Constraints: Sketcher modeler is used to create constraints.
  • Annotations: Drafting modeler is used to create all elements containing text information: dimensions, texts, roughness symbols, welding symbols.
  • Dress up: Drafting modeler is used to create Area fill, axis lines, center lines, arrows and threads.

2D Layout for 3D Design Product Prerequisites

The 2D Layout application is prerequisite to develop an application based on 2DLayout modeler.

Several frameworks are necessary to create a CAA application based on the 2D Layout for 3D Design application:

  • Drafting2DLInterfaces: This framework is dedicated to 2D Layout for 3D Design structuring objects (2D Layout Root,2D Layout Sheet and 2D Layout View)
  • DraftingInterfaces: This framework contains all drafting API except Structuring Objects, 2D geometry, constraint objects.
  • SketcherInterfaces: This framework is dedicated to manage sketcher component: 2D geometry and constraint objects.

2D Layout Structuring Objects

When a 2DL Layout Root is created by using CAT2DLayoutServices Toolbox defined in Drafting2DLInterfaces framework, the following objects are created at the same time:

  • the 2D Layout root
  • The first sheet.
  • A main view aggregated by the first sheet.
  • A background view aggregated by the first sheet.

CAT2DLayoutServices toolbox contains services to:

  • Retrieves the list of available Drawing standards in the environment.
  • Retrieves the list of the sheet style for a given standard.
  • Retrieves the 2D Layout from a given Mechanical Part.
  • Creates a 2D Layout from a Mechanical Part.
