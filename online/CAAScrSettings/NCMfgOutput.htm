NCMfgOutput Setting Repository
Quick Reference

NCMfgOutput

Role: Lists the settings of the operations used in the Machining applications

Detailed role:The operation machining settings are useful to manages options on manufacturing operations

The NCMfgOutput repository has the following characteristics:

  • Primary Tab Page: /Manufacturing/Operation
  • Repository File: NCMfgOutput
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

NCMfgOutput setting repository attributes:

  • MOUpdate
    Role: Do not generate NC code when MO is not uptodate
    Detailed role: Do not generate NC code when MO is not uptodate
    • Type: boolean
    • Default value: false
  • MinCollisionCheck
    Role: Defines the minimum length for collision detection
    Detailed role: Defines the minimum length for collision detection
    • Type: double
    • Default value: 0.0
    • Default Value Role: By default, the minimum length collision detection is set to 0.
  • NCCodeDirectory
    Role: Defines the path for storing the NC code output
    Detailed role: Defines the path to store the NC code output
    • Type: URN
    • Default value: UNIX: ${CATTemp} Windows: ${CATTemp}
  • NCDocDirectory
    Role: Defines the path where the NC documentation files are generated
    Detailed role: Defines the path where the NC documentation files are generated
    • Type: URN
    • Default value: UNIX: ${CATTemp} Windows: ${CATTemp}
  • NCFileExtension
    Role: Defines the extension name of generated NC code files
    • Type: string
    • Default value: CATNCCode
  • OriginFixedOnPart
    Role: Machining Axis Origin
    Detailed role: Fixed on the part after table rotation
    • Type: boolean
    • Default value: false
  • PPPath
    Role: Defines the path where the Post Processors are located
    Detailed role: Defines the path where the Post Processors are located
    • Type: URN
    • Default value: UNIX: ${CATStartupPath}/Manufacturing Windows: ${CATStartupPath}\Manufacturing
  • PackTP
    Role: Store ToolPath at
    Detailed role: Choose the location to store Toolpath
    • Type: string
    • Default value: Document
    • Default Value Role: By default the tool path is in the current document
    • Values:
      Document Store tool path in the current document
      ExternalFile Store tool path in an external file
  • PackTPDirectory
    Role: Store tool path result at
    Detailed role: Choose the location to store tool path
    • Type: string
    • Default value: UNIX: ${CATTemp} Windows: ${CATTemp}
  • PostPro
    Role: Choose the Post Processor company
    Detailed role: Choose the Post Processor company
    • Type: string
    • Default value:
    • Default Value Role: By default no company preference is done
    • Values:
      CENIT CENIT company selection
      IMS IMS company selection
      ICAM ICAM company selection
  • StoreContactPoints
    Role: Store contact points in toolpath during its computation
    Detailed role: Store contact points in toolpath during its computation
    • Type: boolean
    • Default value: true
  • ToolOutputPoint
    Role: Chooses the reference point to compute toolpath with
    Detailed role: Chooses the reference point to compute toolpath with
    • Type: integer
    • Default value: 1
    • Default Value Role: By default the tool output point is the tool tip
    • Values:
      1 Tool tip
      2 Tool center
      3 Tool center for ball end tools
  • ToolPathEditorAccess
    Role: Allows Tool Path edition
    Detailed role: Allows Tool Path edition
    • Type: boolean
    • Default value: true
  • VideoDirectory
    Role: Store Video result at
    Detailed role: Choose the location to store Video
    • Type: string
    • Default value: UNIX: ${CATTemp} Windows: ${CATTemp}
