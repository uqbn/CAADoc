DpmStrSettings Setting Repository
Quick Reference

DpmStrSettings

Role: Features tab page

Detailed role:Features tab page

The DpmStrSettings repository has the following characteristics:

  • Primary Tab Page: /DPManufacturing/DNBDpmStructureWkb/Features
  • Repository File: DNBDpmStructure
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

DpmStrSettings setting repository attributes:

  • AddedMaterialMarginConfiguration
    Role: Margin to be added:
    • Type: integer
    • Default value: 1
  • AddedMaterialMarginSize
    Role: Margin to be added:
    • Type: double
    • Default value: 10
  • AlignmentLineExtremity
    Role: Default extremity
    • Type: short integer
    • Default value: 1
    • Values:
      0 None
      1 Start extremity only
      2 End extremity only
      3 Both extremities
  • AlignmentLineLength
    Role: Length of the alignment mark:
    • Type: double
    • Default value: .150
  • AlignmentLineOffset
    Role: Offset from attachment line extremities:
    • Type: double
    • Default value: .150
  • AttachmentSmartNamingAttr
    Role: Attachment line Smart naming
    Detailed role: the label contain the attached product name
    • Type: short integer
    • Default value: 1
    • Values:
      1 TRUE
      0 FALSE
  • BendingCurveAttribute
    Role: Generate curves
    • Type: short integer
    • Default value: 1
    • Values:
      1 TRUE
      0 FALSE
  • DSTMultipatchAttr
    Role: Treat patches of the plate individually
    • Type: boolean
    • Default value: false
  • EdgePrepType
    Role: Edge Prep Type
    • Type: integer
    • Default value: 0
    • Values:
      0 None
      1 Square
      2 Single_Bevel
      3 Single_Bevel_with_Shoulder
      4 Single_J
      5 DoubleBevel
      6 Double_Bevel_with_Shoulder
      7 Double_J
      8 Other Edge Prep.
  • FTATextGenAttr
    Role: To Generate 3D FTA text or to generate normal text
    • Type: boolean
    • Default value: false
  • FitUpMarginConfiguration
    Role: Margin to be added:
    • Type: integer
    • Default value: 1
  • FitUpMarginSize
    Role: Margin to be reduced by:
    • Type: double
    • Default value: 1
  • GenerateTabbedHolesWithLeadInOut
    Role: Generate tabbed holes with Lead In/Out
    • Type: short integer
    • Default value: 1
    • Values:
      1 TRUE
      0 FALSE
  • LeadInLength
    Role: Lead In Length
    • Type: integer
    • Default value: 10
  • LeadInLengthConfiguration
    Role: Lead in length:
    • Type: integer
    • Default value: 1
  • LeadOutLength
    Role: Lead Out Length
    • Type: integer
    • Default value: 10
  • LeadOutLengthConfiguration
    Role: Lead Out Length:
    • Type: integer
    • Default value: 1
  • LeadSide
    Role: Lead Side
    • Type: integer
    • Default value: 0
    • Values:
      0 None
      1 Left
      2 Right
  • NestingPackageType
    Role: Nesting Package
    • Type: integer
    • Default value: 0
    • Values:
      0 ALMA
      1 OPTINEST
  • OpeningPrepType
    Role: Opening Prep Type
    • Type: integer
    • Default value: 0
    • Values:
      0 None
      1 Cut
      2 Marked
  • PlateTemplate_RollLinesAttr
    Role: Number of roll lines:
    • Type: integer
    • Default value: 1
  • ProfileTemplate_ComboExtremityAttr
    Role: Add to
    • Type: integer
    • Default value: 0
    • Values:
      0 None
      1 Start
      2 End
      3 Both
  • ProfileTemplate_MarkZoneClearanceAttr
    Role: Marking zone clearance:
    • Type: double
    • Default value: 5
  • ProfileTemplate_MinOverLapAttr
    Role: Minimum overlap:
    • Type: double
    • Default value: 450
  • ProfileTemplate_OverLengthAttr
    Role: Over Length
    • Type: double
    • Default value: 55
  • ProfileTemplate_StepValueAttr
    Role: Step value:
    • Type: double
    • Default value: 300
  • TabWidth
    Role: Tab Width
    • Type: integer
    • Default value: 10
  • TabWidthConfiguration
    Role: Tab width:
    • Type: integer
    • Default value: 1
