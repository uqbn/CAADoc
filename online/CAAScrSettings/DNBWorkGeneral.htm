DNBWorkGeneral Setting Repository
Quick Reference

DNBWorkGeneral

Role: Tools Options of DPM Work - General

The DNBWorkGeneral repository has the following characteristics:

  • Primary Tab Page: /DPManufacturing/DNBWork/General
  • Repository File: DNBWorkGeneral
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1
  • Setting controller:
    • Framework_CPP: DNBWorkInterfaces
    • Framework_IDL: DNBWorkInterfaces
    • Interface_CPP: DNBIWKIWorkGeneralSettingAtt
    • Interface_IDL: DNBIAWKIWorkGeneralSettingAtt
    • LateType: DNBWKIWorkGeneralSettingCtrl

DNBWorkGeneral setting repository attributes:

  • AnnotationActivity
    Role: Specifies if Image Capturing will be done for Annotation Activities
    Detailed role: Specifies if Image Capturing will be done for Annotation Activities. Value is 1 for capture 0 for no capture.
    • Type: URN
    • Default value: 1
  • DNBWINumColumn
    Role: Specifies number of WI elements to be shown in 3D
    Detailed role: Specifies number of WI elements to be shown in 3D. It is represents the dimension of DNBWINumColumnList.
    • Type: unsigned integer
    • Default value: 4
  • DNBWINumColumnList
    Role: Indices of WI elements that should be shown in 3D
    Detailed role: Variable Array. Indices of WI elements: 0-Fixed Text, 1-Variable Text , 2-HyperLink, 3-Parameters
    • Type: integer
    • Default value: 3
  • MoveActivity
    Role: Specifies if Image Capturing will be done for Move Activities
    Detailed role: Specifies if Image Capturing will be done for Move Activities. Value is 1 for capture 0 for no capture.
    • Type: URN
    • Default value: 1
  • StyleSheetPath
    Role: Path of Style Sheet for formatting WIs
    Detailed role: Path of Style Sheet for formatting WIs - Specify absolute file path.
    • Type: URN
    • Default value: \intel_a\docs\styles\WorkInstructions.xsl
  • TextActivity
    Role: Specifies if Image Capturing will be done for Text Activities
    Detailed role: Specifies if Image Capturing will be done for Text Activities. Value is 1 for capture 0 for no capture.
    • Type: URN
    • Default value: 0
  • ViewPointActivity
    Role: Specifies if Image Capturing will be done for Viewpoint Activities
    Detailed role: Specifies if Image Capturing will be done for Viewpoint Activities. Value is 1 for capture 0 for no capture.
    • Type: URN
    • Default value: 1
  • VisibilityActivity
    Role: Specifies if Image Capturing will be done for Visibility Activities
    Detailed role: Specifies if Image Capturing will be done for Visibility Activities. Value is 1 for capture 0 for no capture.
    • Type: URN
    • Default value: 1
