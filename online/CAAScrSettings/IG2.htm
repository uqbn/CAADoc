IG2 Setting Repository
Quick Reference

IG2

Role: Handling of IGES 2D Data Exchange Import and Export parameters

Detailed role:Handling of IGES 2D Data Exchange Import and Export parameters

The IG2 repository has the following characteristics:

  • Primary Tab Page: /General/Compatibility/IGES 2D
  • Repository File: IG2
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1
  • Root tab page /x reused as /x

IG2 setting repository attributes:

  • ExportMode
    Role: Export Mode semantic, strucured or graphic
    • Type: integer
    • Default value: 2
    • Values:
      0 Export semantic.
      1 Export structured.
      2 Export graphic.
  • ExportSheets
    Role: Which sheets to export
    • Type: integer
    • Default value: 0
    • Values:
      0 Export All sheets. One File is generated per none empty sheet.
      1 Export only the current sheet in a single file.
  • ImportDestinationView
    Role: In which view put the result
    • Type: integer
    • Default value: 0
    • Values:
      0 Put the import result into Working Views.
      1 Put the import result into Background.
  • ImportDftStandard
    Role: name of the Drafting standard that defines the standard of the resulting Drawing
    • Type: string
    • Default value: ISO
  • ImportDimensions
    Role: How to import dimensions
    • Type: integer
    • Default value: 1
    • Values:
      0 Convert dimensions as dimensions
      1 The Convert dimensions as details
      2 Convert dimensions as geometry
  • ImportEndPoints
    Role: When create end points
    • Type: integer
    • Default value: 0
    • Values:
      0 Never create end points.
      1 Create end points only for few entities.
      2 Always create end points.
  • ImportUnit
    Role: the unit of the input file
    • Type: integer
    • Default value: 5
    • Values:
      0 The unit of the input file is Millimeter
      1 The unit of the input file is Centimeter
      2 The unit of the input file is Meter
      3 The unit of the input file is Inch
      4 The unit of the input file is Foot
      5 The unit of the input file is determined Automatically
