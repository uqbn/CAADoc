AssemblyGeneral Setting Repository
Quick Reference

AssemblyGeneral

Role: Setting to manage Assembly Design (General) options

Detailed role:Assembly Design (General) setting contains all options necessary to Assembly Design (General) commands

The AssemblyGeneral repository has the following characteristics:

  • Primary Tab Page: /Mechanical Design/Assembly Design/General
  • Repository File: Assembly
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

AssemblyGeneral setting repository attributes:

  • RealUpdtStatus
    Role: Attribute to manage Compute Update at Open mode (automatic or manual)
    Detailed role: Automatically computes exact update status, or highlight update icon on opening a document
    • Type: integer
    • Default value: 0
    • Default Value Role: By default manually compute exact update status
    • Values:
      0 Manually compute exact update status
      1 Automatically computes exact update status
  • UpdAuto
    Role: Attribute to manage IsUpToDate mode (automatic or manual)
    Detailed role: Automatically computes real update status, or highlight update icon whenever a modification is performed
    • Type: integer
    • Default value: 0
    • Default Value Role: By default update icon gets highlighted whenever a modification is performed
    • Values:
      0 Update icon gets highlighted whenever a modification is performed
      1 Automatically computes real update status
  • UpdDepth
    Role: Attribute to manage update propagation depth (active level or all the levels).
    Detailed role: Decides propagation depth of update.
    • Type: integer
    • Default value: 0
    • Default Value Role: By default Update is done for all the levels.
    • Values:
      0 Update is done for only for active level.
      1 Update is done for all the levels.
  • WarningWhenMovingFixT
    Role: Attribute managing the extension of a move to the components involved in a FixTogether
    Detailed role: When a component involved in a FixTogether is moved, same move may be applied to all other components involved in the same FixTogether, according to this attribute's value
    • Type: integer
    • Default value: 0
    • Default Value Role: By default Move is never extended
    • Values:
      0 Never extend move to all components involved in a FixTogether
      1 Ask question to extend move to all component involved in a FixTogether
      2 always extend move to all component involved in a FixTogether
  • autoSwitchMode
    Role: Access to geometry
    Detailed role: Mode to decide if automatic switch to Design mode
    • Type: integer
    • Default value: 0
    • Default Value Role: By default automatic switch to Design mode
    • Values:
      0 Automatic switch to Design mode available
      1 Automatic switch to Design mode unavailable
