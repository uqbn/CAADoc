DNBDpmCompareTool

Role: Tab page to handle the Compare Tool command Setting parameters

The DNBDpmCompareTool repository has the following characteristics:

  • Primary Tab Page: /DPManufacturing/Compare Tool
  • Repository File: DNBDpmCompareTool
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1
  • Setting controller:
    • Framework_CPP: DPMSettingsInterfaces
    • Framework_IDL: DPMSettingsInterfaces
    • Interface_CPP: DNBIDSICompareToolSettingAtt
    • Interface_IDL: DNBIACompareToolSettingAtt
    • LateType: DNBDSICompareToolSettingCtrl
