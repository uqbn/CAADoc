DNBMfgLayoutSettings Setting Repository
Quick Reference

DNBMfgLayoutSettings

Role: MRL WorkBench Tab page

Detailed role:Settings to handle MRL Attributes

The DNBMfgLayoutSettings repository has the following characteristics:

  • Primary Tab Page: /IGRIP/ManufacturingLayout
  • Repository File: DNBMfgLayoutSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1
  • Setting controller:
    • Framework_CPP: DNBManufacturingLayoutItf
    • Framework_IDL: DNBManufacturingLayoutItf
    • Interface_CPP: DNBIMltMfgLayoutSettingAtt
    • Interface_IDL: DNBIAMfgLayoutSettingAtt
    • LateType: DNBMltMfgLayoutSettingCtrl

DNBMfgLayoutSettings setting repository attributes:

  • FillColorStatus
    Role: Footprint - Fill color to the footprint
    • Type: unsigned integer
    • Default value: 0
  • FootprintExistence
    Role: Footprint - No Footprint Representation
    • Type: unsigned integer
    • Default value: 1
  • FootprintPickable
    Role: Footprint - Pick footprint
    • Type: unsigned integer
    • Default value: 0
  • GridNumberStatus
    Role: Grid - Show Grid numbering
    • Type: unsigned integer
    • Default value: 0
  • GridOnTop
    Role: Grid - Grid always on top
    • Type: unsigned integer
    • Default value: 0
  • ShowFootprint
    Role: Footprint - Show Footprints
    • Type: unsigned integer
    • Default value: 0
  • ShowGrid
    Role: Show Grid By Default
    Detailed role: Grid - Show Grid by Default
    • Type: unsigned integer
    • Default value: 0
  • ShowResourceName
    Role: Footprint - Show Resource Name on footprint
    • Type: unsigned integer
    • Default value: 0
