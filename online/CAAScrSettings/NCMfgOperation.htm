NCMfgOperation Setting Repository
Quick Reference

NCMfgOperation

Role: Lists the settings of the operations used in the Machining applications

Detailed role:The operation machining settings are useful to manages options on manufacturing operations

The NCMfgOperation repository has the following characteristics:

  • Primary Tab Page: /Machining/Operation
  • Repository File: NCMfgOperation
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

NCMfgOperation setting repository attributes:

    • Default value: true

    Role: Duplicate Geometry links when copying
    Detailed role: Duplicate Geometry links when copying
    • Type: boolean
    • Default value: true
  • CreateDefTool
    Role: Use default tool
    Detailed role: Use default tool

    • Default value: true

    Role: Display toolpath of operation in current Part-Operation
    Detailed role: Display toolpath of operation in current Part-Operation

    • Default value: false
  • MPInstant_TLAbsPos
    Role: Keep absolute position of tool axis and machining direction
