ElectricalHarnessInstallation Setting Repository
Quick Reference

ElectricalHarnessInstallation

Role: Define electrical harness management settings

Detailed role:Define electrical harness management settings

The ElectricalHarnessInstallation repository has the following characteristics:

  • Primary Tab Page: /Equipment & Systems/Electrical Harness Installation/Harness Management
  • Repository File: ElectricalHarnessInstallation
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

ElectricalHarnessInstallation setting repository attributes:

  • AccuracyState
    Role: Setting to manage the balance between performance and accuracy
    Detailed role: Setting to manage the balance between performance and accuracy for the FLEX algorithm
    • Type: integer
    • Default value: 1
    • Min value: 1
    • Max value: 10
  • AllowSystem
    Role: CheckButton to allow System to use a Min Bend Radius for Curve Computation
    Detailed role: Setting to allow System to use a Min Bend Radius for Curve Computation
    • Type: integer
    • Default value: 524288
    • Default Value Role: By default, do not allow the system to use a smaller bend radius value
    • Values:
      1 Allow the system to use a smaller bend radius value
      524288 Do not allow the system to use a smaller bend radius value
  • AutoLinkState
    Role: Activate/Deactivate automatic link
    • Type: integer
    • Default value: 1
    • Values:
      0 Deactivate automatic link to new bundle segment
      1 Activate automatic link to new bundle segment
  • AutoUnLinkState
    Role: Settings to remove geometrical link at unlink
    • Type: integer
    • Default value: 1
    • Values:
      0 Do not remove geometrical link at unlink
      1 Remove geometrical link at unlink
  • AvoidWarningMessageRepetition
    Role: Check Button to avoid repetition of warning message when building curve
    Detailed role: Avoids repetition of warning message.i.e. when curve is broken OR when curve is built with dichotomy
    • Type: integer
    • Default value: 0
    • Default Value Role: By default the warning message is not avoided
    • Values:
      0 Don't avoid repetition of warning message
      1 Avoid repetition of warning message
  • BNSCollapsed
    Role: Check Button to warn user when BundleSegment is collapsed
    Detailed role: Warns the user when BundleSegment is collapsed
    • Type: integer
    • Default value: 1
    • Default Value Role: By default user gets warning when BundleSegment is collapsed
    • Values:
      0 Dont Warn user when BundleSegment is collapsed
      1 Warn user when BundleSegment is collapsed
  • BNSColorChange
    Role: Check Button to change color of BundleSegment as well
    Detailed role: Changes the color of BundleSegment as well
    • Type: integer
    • Default value: 0
    • Default Value Role: By default the color of BundleSegment does not change
    • Values:
      0 Will not Change the color of BundleSegment as well
      1 Will Changes the color of BundleSegment as well
  • BNSColorToSet
    Role: Color to Set.
    • Type: unsigned integer
    • Default value: 0
    • Default Value Role: Red in RGB order
    • Min value: 0
    • Max value: 255
  • ChoixAlgo
    Role: Setting to manage the algorithm used for branch routing
    Detailed role: Setting to manage the algorithm used for branch routing
    • Type: integer
    • Default value: 0
    • Default Value Role: By default the formal algorithm is used for the branch routing
    • Values:
      0 The standard algorithm is used for the branch routing
      1 The FLEX algorithm is used for the branch routing
  • CombolinkoptionsState
    Role: Setting for keep link with selected object
    • Type: integer
    • Default value: 0
    • Min value: 0
    • Max value: 2
    • Step: 1
  • ContextualLinkCheckState
    Role: Setting to forbid selection outside current geometrical bundle
    • Type: integer
    • Default value: 0
    • Values:
      0 Do not forbid the selection outside current geometrical bundle
      1 Forbid the selection outside current geometrical bundle
  • CurrentExtremityColor
    Role: Define bundle segment color
    • Type: unsigned integer
    • Default value: 255
    • Default Value Role: White in RGB order
    • Min value: 0
    • Max value: 255
  • CurveNotElectrical
    Role: Check Button to warn user when Curve is not Electrical
    Detailed role: Warns the user when the Curve is not Electrical
    • Type: integer
    • Default value: 1
    • Default Value Role: By default user gets warning when Curve is not Electrical
    • Values:
      0 Dont Warn user when Curve is not Electrical
      1 Warn user when Curve is not Electrical
  • DefaultNameState
    Role: Setting to manage bundle segment naming
    • Type: integer
    • Default value: 1
    • Values:
      1 Use bundle segment naming rule
      0 Use parent-bundle segment naming rule
  • FlexToStandard
    Role: Checks the option of curve build
    • Type: integer
    • Default value: 0
  • GeometricLinkState
    Role: Activate/Deactivate geometrical link
    • Type: integer
    • Default value: 1
    • Values:
      0 Deactivate geometrical link between segments
      1 Activate geometrical link between segments
  • HideInternal
    Role: Hide internal harness geometry
    • Type: integer
    • Default value: 1
    • Values:
      0 Do not hide internal harness geometry
      1 Hide internal harness geometry
  • KeepCustomizedProdNameForGBNState
    Role: Use customised display for geometrical bundle naming rule
    • Type: integer
    • Default value: 0
    • Values:
      0 Do not use customized display for geometrical bundle nameing rule
      1 Use customized display for geometrical bundle nameing rule
  • KeepNotUsedState
    Role: Setting to manage the unused branch points
    Detailed role: Setting to manage the unused branch points
    • Type: integer
    • Default value: 1
    • Default Value Role: By default an unused branch point is not deleted
    • Values:
      1 An unused branch point is not deleted
      0 An unused branch point is prompted for deletion
  • LocalSlackState
    Role: Check Button to display labels for Local Slack
    Detailed role: Check Button to display labels for Local Slack
    • Type: integer
    • Default value: 0
    • Default Value Role: By default Labels should not be displayed
    • Values:
      0 Do not display the labels for Local Slack
      1 Display the labels for Local Slack
  • MandatoryGBNState
    Role: Define if the geometrical bundle is mandatory
    Detailed role: Define if the geometrical bundle is mandatory
    • Type: integer
    • Default value: 1
    • Values:
      0 Geometrical bundle is not mandatory
      1 Geometrical bundle is mandatory
  • NumberOfDocAssistanceState
    Role: Work with one multi-branchable per geometrical bundle
    • Type: integer
    • Default value: 0
    • Values:
      0 Do not work with one multi-branchable per geometrical bundle
      1 Work with one multi-branchable per geometrical bundle
  • PointRepColorToSet
    Role: Define symbol color for points when adding local slack
    • Type: unsigned integer
    • Default value: 0
    • Default Value Role: Red in RGB order
    • Min value: 0
    • Max value: 255
  • PointRepSymbolToSet
    Role: Define point representation of symbol used for points when adding local slack
    • Type: integer
    • Default value: 1
    • Default Value Role: Cross
    • Values:
      1 Cross
      2 Plus
      3 Circle
      4 Double Circle
      5 Dot
      6 Full Square
      7 Star
      8 Small Full Square
      9 Small Dot
  • PointSymbolChkBtnState
    Role: Check Button to display specific symbol for points when adding local slack
    Detailed role: Displays specific symbol for points when adding local slack
    • Type: integer
    • Default value: 0
    • Default Value Role: By default specific symbol should not be displayed
    • Values:
      0 Do not use specific symbol for local slack display
      1 Use specific symbol for local slack display
  • ProfileTypeState
    Role: Settings for bundle segment profile
    Detailed role: Choose 'Circular profile only', to work with circular section bundle segments only.
    Choose 'Any profile', to work with bundle segment sections of any type.
    • Type: integer
    • Default value: 1
    • Default Value Role: By default, use circular profile only
    • Values:
      1 Use circular profile only
      2 Use any type of profile
  • ProfileValueState
    Role: Settings for bundle segment profile
    Detailed role: Choose 'Circular profile only', to work with circular section bundle segments only.
    Choose 'Any profile', to work with bundle segment sections of any type.
    • Type: integer
    • Default value: 1
    • Default Value Role: By default, keep the greatest value constant is set
    • Values:
      1 Keep the greatest value constant
      2 Keep the smallest value constant
      3 Keep the ratio constant
      4 Do not recompute values
  • RatioCLMode
    Role: Setting to manage the accuracy of the final branch length in constrained length mode
    Detailed role: Setting to manage the accuracy of the final branch length in constrained length mode
    • Type: double
    • Default value: 1
    • Default Value Role: By default the tolerance is set to one percent
  • SptPubVisuModeState
    Role: Settings to use publications to recognize adaptive supports
    Detailed role: This option will be used during selective loading and extract
    If you activate it, only support with publication 'ELEC-SPPORT-RADIUS' will be considered as adaptive.
    Its highly recommneded to use it because it will make you save time when loading large assemblies containing adaptive supports.
    • Type: integer
    • Default value: 1
    • Values:
      0 Do not use publications to recognize adaptive supports
      1 Use publications to recognize adaptive supports
  • UserBendRadNotRespected
    Role: Check Button to warn user when User Bend Radius is not Respected
    Detailed role: Warns the user when User Bend Radius is not Respected
    • Type: integer
    • Default value: 1
    • Default Value Role: By default user gets warning when User Bend Radius is not Respected
    • Values:
      0 Dont Warn user when User Bend Radius is not Respected
      1 Warn user when User Bend Radius is not Respected
  • WarnOnBranchCollapse
    Role: Check Button to warn user when Branch collapses
    Detailed role: Warns the user when the Branch collapses i.e. BendRadius is less than Segment Radius
    • Type: integer
    • Default value: 1
    • Default Value Role: By default user gets a warning when Branch collapses
    • Values:
      0 Dont Warn user when Branch is collapsed
      1 Warn user when Branch is collapsed
