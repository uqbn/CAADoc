DNBMSDResUtilSettings Setting Repository
Quick Reference

DNBMSDResUtilSettings

Role: Provides Options to manage the MSD settings parameters

The DNBMSDResUtilSettings repository has the following characteristics:

  • Primary Tab Page: /DPManufacturing/DNBMSDWkb/General
  • Repository File: DNBMSDResUtilSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1
  • Setting controller:
    • Framework_CPP: DPMSettingsInterfaces
    • Framework_IDL: DPMSettingsInterfaces
    • Interface_CPP: DNBIMSDResUtilSettingAtt
    • Interface_IDL: DNBIAResUtilSettingAtt
    • LateType: DNBMSDResUtilSettingCtrl

DNBMSDResUtilSettings setting repository attributes:

  • FirstLevelColor
    Role: The color for the first level of Utilization
    Detailed role: The color for the first level of Utilization
    • Type: integer
    • Default value: 0
  • FirstLevelEnabledCheckButton
    Role: First Level of Utilization
    Detailed role: First Level of Utilization
    • Type: boolean
    • Default value: true
  • FirstLevelGreaterValue
    Role: To set the Value of utilization beyond which the color of the utilization is set to the value set in the FirstLevelColor combo
    Detailed role: To set the Value of utilization beyond which the color of the utilization is set to the value set in the FirstLevelColor combo
    • Type: integer
    • Default value: 100
  • FourthLevelColor
    Role: The color for the fourth level of Utilization
    Detailed role: The color for the fourth level of Utilization
    • Type: integer
    • Default value: 0
  • FourthLevelEnabledCheckButton
    Role: Third level of Utilization
    Detailed role: Third level of Utilization
    • Type: boolean
    • Default value: true
  • FourthLevelLessEqualValue
    Role: Fourth Level Less than Equal Value
    Detailed role: Fourth Level Less than Equal Value
    • Type: integer
    • Default value: 75
  • ProductFlowMode
    Role: Option to determine if the Product Flow is to be considered
    Detailed role: This option determines if the Product Flow is to be considered. The Product flow is considered if checked on and not considered otherwise
    • Type: boolean
    • Default value: false
  • ResUtilEnabledCheckButton
    Role: To Caculate and display the utilization of Logical Resources
    Detailed role: If checked on calculates and displays the utilization of Logical Resources
    • Type: boolean
    • Default value: true
  • SecondLevelColor

    Detailed role: The color for the second level of Utilization
    • Type: integer
    • Default value: 125
  • SecondLevelEnabledCheckButton

  • SecondLevelGreaterValue

    Detailed role: To set the Value of utilization beyond which the color of the utilization is set to the value set in the SecondLevelColor combo
    • Type: integer
    • Default value: 90
  • SecondLevelLessEqualValue
    Role: Second Level Less than Equal Value
    Detailed role: Second Level Less than Equal Value
    • Type: integer
    • Default value: 100
  • TaktTimeSec

    Detailed role: Displays the Takt Time in Seconds
    • Type: double
