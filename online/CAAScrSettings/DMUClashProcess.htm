DMUClashProcess Setting Repository
Quick Reference

DMUClashProcess

Role: Setting to manage the XML export of Clash results

Detailed role:Setting to manage the XML export of Clash results

The DMUClashProcess repository has the following characteristics:

  • Primary Tab Page: /Digital Mockup/DMU Space Analysis/DMU Clash - Process
  • Repository File: ClashPublish
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

DMUClashProcess setting repository attributes:

  • AggregateFiles
    Role: Aggregate files in a single folder
    Detailed role: Aggregate files in a single folder
    • Type: integer
    • Default value: 0
  • ColorBackground
    Role: Background is colored or transparent
    Detailed role: Background is colored or transparent
    • Type: integer
    • Default value: 0
  • ExportCProcess
    Role: XML export for Clash
    Detailed role: XML export for Clash
    • Type: integer
    • Default value: 0
  • FederatedOutput
    Role: Federated output
    Detailed role: Federated output
    • Type: integer
    • Default value: 0
  • LongFileNames
    Role: Support for long filenames
    Detailed role: Support for long filenames
    • Type: integer
    • Default value: 0
  • PictureGenerated
    Role: Picture is generated
    Detailed role: Picture is generated
    • Type: integer
    • Default value: 0
  • PictureZoom
    Role: Picture is zoomed
    Detailed role: Picture is zoomed
    • Type: integer
    • Default value: 0
  • PublishDefDir
    Role: Path of the directory where the xml results will be generated
    Detailed role: Path of the directory where the xml results will be generated
    • Type: string
    • Default value: ""
  • XslOutput
    Role: Browser opened automatically
    Detailed role: Browser opened automatically
    • Type: integer
    • Default value: 0
  • XslTemplate
    Role: Path of style sheet for the XML dump
    Detailed role: Path of style sheet for the XML dump
    • Type: string
    • Default value: ""
