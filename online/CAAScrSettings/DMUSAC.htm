DMUSAC Setting Repository
Quick Reference

DMUSAC

Role: Setting to handle the DMU navigator parameters

The DMUSAC repository has the following characteristics:

  • Primary Tab Page: /Digital Mockup/DMU Navigator
  • Repository File: CATSACRepository
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

DMUSAC setting repository attributes:

  • InsertLevel
    Role: Level for insert applicative data
    • Type: integer
    • Default value: 0
  • InsertMode
    Role: Import of applicative data
    • Type: integer
    • Default value: 0
    • Values:
      0 No import of the applicative data
      1 Import applicative data with user prompt
      2 Import applicative data automatically
