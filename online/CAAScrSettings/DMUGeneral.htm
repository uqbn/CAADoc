DMUGeneral Setting Repository
Quick Reference

DMUGeneral

Role: Settings related to DMU Navigator

The DMUGeneral repository has the following characteristics:

  • Primary Tab Page: /Digital Mockup/General
  • Repository File: 4DNavigator
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

DMUGeneral setting repository attributes:

  • DMUClashPreview
    Role: Clash preview
    • Type: integer
    • Default value: 0
  • DMUCut3DPreview
    Role: 3DCut preview
    • Type: integer
    • Default value: 1
  • DMUDistancePreview
    Role: Distance preview
    • Type: integer
    • Default value: 0
  • DMUFreeSpacePreview
    Role: Free space preview
    • Type: integer
    • Default value: 1
  • DMUGroupPreview
    Role: Group preview
    • Type: integer
    • Default value: 1
  • DMUGroupPreviewHiddenObjectsColor
    Role: Group preview hidden object color
    • Type: unsigned integer
    • Default value: 255
    • Default Value Role: Default is No Color
    • Min value: 0
    • Max value: 255
  • DMUGroupPreviewHiddenObjectsDisplayMode
    Role: Group preview hidden object display mode
    • Type: integer
    • Default value: 0
  • DMUGroupPreviewHiddenObjectsLowInt
    Role: Low intensity Group preview
    • Type: integer
    • Default value: 0
  • DMUGroupPreviewHiddenObjectsOpacity
    Role: Group preview hidden object opacity
    • Type: integer
    • Default value: 300
  • DMUGroupPreviewHiddenObjectsPick
    Role: Group preview hidden object pickable
    • Type: integer
    • Default value: 1
  • DMUMergerPreview
    Role: Merger preview
    • Type: integer
    • Default value: 1
  • DMUOffsetPreview
    Role: Offset preview
    • Type: integer
    • Default value: 1
  • DMUSectionPreview
    Role: Section preview
    • Type: integer
    • Default value: 0
  • DMUShuttlePreview
    Role: Shuttle preview
    • Type: integer
    • Default value: 1
  • DMUSilhouettePreview
    Role: Silhouette preview
    • Type: integer
    • Default value: 1
  • DMUSimplifPreview
    Role: Simplification preview
    • Type: integer
    • Default value: 1
  • DMUSweptVolPreview
    Role: Swept volume preview
    • Type: integer
    • Default value: 1
  • DMUThicknessPreview
    Role: Thickness preview
    • Type: integer
    • Default value: 1
  • DMUVibrationVolPreview
    Role: Vibration volume preview
    • Type: integer
    • Default value: 1
  • DMUWrappingPreview
    Role: Wrapping preview
    • Type: integer
    • Default value: 1
