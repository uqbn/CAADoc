DMUClashDetailedComputation Setting Repository
Quick Reference

DMUClashDetailedComputation

Role: Setting to manage the interference computation specifications

Detailed role:Setting to manage the interference computation specifications

The DMUClashDetailedComputation repository has the following characteristics:

  • Primary Tab Page: /Digital Mockup/DMU Space Analysis/DMU Clash - Detailed Computation
  • Repository File: Clash
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

DMUClashDetailedComputation setting repository attributes:

  • ModeOfClashResult
    Role: Mode of clash result
    Detailed role: Mode of clash result
    • Type: integer
    • Default value: 1
    • Default Value Role: By default, the mode of result is Curve
    • Values:
      1 Curve
      0 None
  • ModeOfClearContResult
    Role: Mode of clearance and contact result
    Detailed role: Mode of clearance and contact result
    • Type: integer
    • Default value: 1
    • Default Value Role: By default, the mode of clearance and contact result is Triangle
    • Values:
      1 Triangle
      2 Surface
      0 None
  • ModeOfLevelDetail
    Role: Level Of Detail
    Detailed role: Level Of Detail
    • Type: integer
    • Default value: 1
    • Default Value Role: By default, the level of detail is Element
    • Values:
      1 Element
      2 Product
  • ModeOfNumericResult
    Role: Mode of numeric result
    Detailed role: Mode of numeric result
    • Type: integer
    • Default value: 1
    • Default Value Role: By default, the mode of numeric result is Penetration Depth or Minimum Distance
    • Values:
      1 PenDepthMinDist
      0 None
  • ModeOfSurfElement
    Role: Consider surfaces for clash computation
    Detailed role: Consider surfaces for clash computation
    • Type: integer
    • Default value: 0
  • SizeOfVoxel
    Role: Value of volxel size
    Detailed role: Value of volxel size
    • Type: double
    • Default value: 5
  • TypeOfIntersectVolume
    Role: Type of Intersection Volume
    Detailed role: Type of Intersection Volume
    • Type: integer
    • Default value: 0
    • Default Value Role: By default, the type of Intersection Volume is None
    • Values:
      1 Voxel
      2 Boolean
      0 None
  • ValueOfDiscrSurface
    Role: Value of discritization
    Detailed role: Value of discritization
    • Type: double
    • Default value: 10
