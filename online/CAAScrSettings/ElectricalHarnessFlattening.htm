ElectricalHarnessFlattening setting repository attributes:

    • Type: string
    • Default value:

    Role: Electrical connection option lock
    • Type: integer
    • Default value: 0

    Role: Electrical connection option

    • Default value: 1
    • Values:
      0 Do not use electrical connection
