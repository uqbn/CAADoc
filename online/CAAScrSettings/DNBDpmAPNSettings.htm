DNBDpmAPNSettings Setting Repository
Quick Reference

DNBDpmAPNSettings

Role: Settings to handle APN Attributes.

Detailed role:Settings to handle APN Attributes.

The DNBDpmAPNSettings repository has the following characteristics:

  • Primary Tab Page: /Digital Process For Manufacturing/DPM - Assembly Process Planner/Commands
  • Repository File: DNBDpmAPNSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

DNBDpmAPNSettings setting repository attributes:

  • DELAPNCheckPartAssignmentInPPRHUB
    Role: Part Assignment - Check part assignment with process in manufacturing Hub
    • Type: string
    • Default value: CATDlgEnable
  • DELAPNCopyProcPartRelId
    Role: Assignment Assistant - Copy Object ID on process part relation
    • Type: string
    • Default value: CATDlgDisable
  • DELAPNDisplayFeatureGeometryIn3DViewer
    Role: Assignment Assistant - Display Fastener Geometry in 3D Viewer
    • Type: string
    • Default value: CATDlgDisable
  • DELAPNMACheckAA
    Role: Support Manufacturing Assembly Assignment in Assignment Assistant
    • Type: string
    • Default value: CATDlgDisable
  • DELAPNMHubProcessName
    Role: Display MHub Configured Process Name in Assignement Assistant
    • Type: string
    • Default value: CATDlgDisable
  • DELAPNMKitCheckAA
    Role: Support Manufacturing Kit Assignment in Assignment Assistant
    • Type: string
    • Default value: CATDlgDisable
  • DELAPNPCPCheck
    Role: Support Process Creates Product (PCP) Relation
    • Type: string
    • Default value: CATDlgDisable
  • DELAPNPFPPCheck
    Role: "Support Process First Processes Product (PFPP) Relation
    • Type: string
    • Default value: CATDlgEnable
  • DELAPNPPPCheck
    Role: Support Process Processes Product (PPP) Relation
    • Type: string
    • Default value: CATDlgDisable
  • DELAPNPRPCheck
    Role: Support Process Removes Product (PRP) Relation
    • Type: string
    • Default value: CATDlgDisable
  • DELAPNPreviewPartInDesignPosition
    Role: Assignment Assistant - Preview parts in design position
    • Type: string
    • Default value: CATDlgDisable
  • DELAPNShowPartNameAsInstanceName
    Role: AST Editor - Show part name as instance name
    • Type: string
    • Default value: CATDlgDisable
  • DELAPNShowRelationsCheck
    Role: "Process Product Relations
    • Type: string
    • Default value: CATDlgDisable
