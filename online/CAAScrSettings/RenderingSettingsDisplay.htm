RenderingSettingsDisplay Setting Repository
Quick Reference

RenderingSettingsDisplay

Role: Real Time Rendering options

Detailed role:Full explanation of the Setting Role

The RenderingSettingsDisplay repository has the following characteristics:

  • Primary Tab Page: /Infrastructure/Real Time Rendering/Display
  • Repository File: RenderingOptionsSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

RenderingSettingsDisplay setting repository attributes:

  • ActiveLightVisuMode
    Role: Set inactive lights display mode
    • Type: integer
    • Default value: 1
    • Values:
      1 No display
      2 Full Display
  • EnvironmentVisuMode
    Role: Set inactive environments display mode
    • Type: integer
    • Default value: 1
    • Values:
      1 No display
      2 Simplified Display
      3 Full Display
  • LightManipMode
    Role: Show the surface normal when positioning lights
    • Type: boolean
    • Default value: false
  • LightVisuMode
    Role: Type of light representation
    • Type: integer
    • Default value: 1
    • Values:
      1 Wireframe
      2 Shaded
  • ReflectionDynamicMode
    Role: Enable update of reflection when viewpoint is changed
    • Type: boolean
    • Default value: false
  • ReflectionDynamicSize
    Role: Set reflection quality when viewpoint is moving
    • Type: integer
    • Default value: 128
  • ReflectionMode
    Role: Enable reflection of the active environment on reflective geometries
    • Type: boolean
    • Default value: false
  • ReflectionPasses
    Role: Set reflection computation passes number
    • Type: integer
    • Default value: 4
  • ReflectionStaticSize
    Role: Set reflection quality when viewpoint is not moving
    • Type: integer
    • Default value: 256
  • ReflectionUpdateAngle
    Role: Set reflection update angle threshold
    • Type: integer
    • Default value: 10
