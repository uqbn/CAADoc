RenderingSettingsGeneral Setting Repository
Quick Reference

RenderingSettingsGeneral

Role: Real Time Rendering options

Detailed role:Full explanation of the Setting Role

The RenderingSettingsGeneral repository has the following characteristics:

  • Primary Tab Page: /Infrastructure/Photo Studio/General
  • Repository File: RenderingOptionsSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

RenderingSettingsGeneral setting repository attributes:

  • EngineInterface
    Role: Set the selected rendering engine
    • Type: string
    • Default value: CATIRdgDefaultEngine
  • EnvironmentSnap
    Role: Enable automatic environment snap to geometry when created
    • Type: boolean
    • Default value: false
  • LightCreationAxis
    Role: Set gravitation direction
    • Type: integer
    • Default value: 2
    • Values:
      0 X
      1 Y
      2 Z
  • LightCreationMode
    Role: Set direction of created lights
    • Type: integer
    • Default value: 0
    • Values:
      0 Vertical
      1 Viewpoint
      2 Gravitational
  • MaterialViewMode
    Role: Force material display when entering workbench
    • Type: boolean
    • Default value: false
  • ViewpointMode
    Role: Set viewpoint mode when entering workbench
    • Type: integer
    • Default value: 1
    • Values:
      0 Parallel
      1 Perspective
      2 Last Saved
  • ViewpointModeActivated
    Role: Enable change of viewpoint type when entering workbench
    • Type: boolean
    • Default value: false
