GBiWDisplay Setting Repository
Quick Reference

GBiWDisplay

Role: These setting Parameters define the default Fastener Display Parameters

Detailed role:These Display setting Parameters can be classified into two categories: A) Display of Fastener Technological Parameters (Fastener Type .etc) B) Display of Fastener Shape Parameters (Fastener Location,.etc)

The GBiWDisplay repository has the following characteristics:

  • Primary Tab Page: /Shape/AutomotiveBiWFastening/Display
  • Repository File: GBiWFastening
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

GBiWDisplay setting repository attributes:

  • BfmBaseZoneMatOrientationShow
    Role: Show/NoShow State of the Symbol
    • Type: boolean
    • Default value: false
  • BfmFastenerNormalOrientationShow
    Role: Show/NoShow State of the Symbol
    • Type: boolean
    • Default value: false
  • BfmKAxisOnlyShow
    Role: FALSE value: I, J and K Axis are visualized
    • Type: boolean
    • Default value: true
  • BfmLocationPointShow
    Role: Show/NoShow State of the Symbol
    • Type: boolean
    • Default value: true
  • BfmMaterialOrientationsShow
    Role: Show/NoShow State of the Symbol
    • Type: boolean
    • Default value: false
  • BfmProjectedPointShow
    Role: Show/NoShow State of the Symbol
    • Type: boolean
    • Default value: false
  • BfmRefAxisOnEachPointShow
    Role: Display fastener axis orientation on each discretization point
    • Type: boolean
    • Default value: false
  • BfmSectionShow
    Role: Show/NoShow State of the Symbol
    • Type: boolean
    • Default value: false
  • BfmStackUpOrderShow
    Role: Show/NoShow State of the Symbol
    • Type: boolean
    • Default value: false
  • BfmTextShow
    Role: Show/NoShow State of the Symbol
    • Type: boolean
    • Default value: false
  • COL_ASSIGNED_SPEC
    Role: Fastener technological parameter assigned to Symbol Color
    • Type: string
    • Default value: PCA
    • Values:
      "PCA" Process Category is assigned to Symbol Color
      "U" Unspecified: no specification is assigned to Symbol
      "NONE" None is assigned to Symbol
  • MARK_ASSIGNED_SPEC
    Role: Fastener technological parameter assigned to Symbol Mark
    • Type: string
    • Default value: NONE
    • Values:
      "ROB" Robustness is assigned to Symbol Mark
      "REG" Regulation is assigned to Symbol Mark
      "FIN" Finish is assigned to Symbol Mark
      "NONE" None is assigned to Symbol Mark
  • MARK_ASSIGNED_THKSCALE
    Role: Mark Thickness Value
    • Type: integer
    • Default value: 1
    • Min value: 1
    • Max value: 3
    • Step: 1
  • SYM_ASSIGNED_SPEC
    Role: Fastener Technological parameter assigned to Symbol
    • Type: string
    • Default value: PTY
    • Values:
      "PTY" Process Type is assigned to Symbol
      "U" Unspecified: no specification is assigned to Symbol
      "NTH" Thickness Count is assigned to Symbol
      "NONE" None is assigned to Symbol
  • SYM_ASSIGNED_THKSCALE
    Role: Symbol Thickness Value
    • Type: integer
    • Default value: 1
    • Min value: 1
    • Max value: 3
    • Step: 1
