Sketcher Setting Repository
Quick Reference

Sketcher

Role: Setting to manage the options of the Sketcher

Detailed role:Sketcher setting contains all options necessary to the commands of the Sketcher

The Sketcher repository has the following characteristics:

  • Primary Tab Page: /Mechanical Design/Sketcher/Sketcher
  • Repository File: Sketcher
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

Sketcher setting repository attributes:

      1 The user can specify distorsion in the grid
