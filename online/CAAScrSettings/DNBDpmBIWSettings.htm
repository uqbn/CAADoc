DNBDpmBIWSettings Setting Repository
Quick Reference

DNBDpmBIWSettings

Role: Settings to handle Fastening Process Planner Attributes.

Detailed role:Settings to handle FPP Attributes.

The DNBDpmBIWSettings repository has the following characteristics:

  • Primary Tab Page: /DPM - Fatening Process Planner/Commands
  • Repository File: DNBDpmBIWSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

DNBDpmBIWSettings setting repository attributes:

  • ApplyAlternateProcAssigment
    Role: Fastener Assignment - Allow assignments to alternate process
    • Type: string
    • Default value: CATDlgDisable
  • ApplyAssignmentInPPRHUB
    Role: Fastener Assignment - Check for assignments in manufacturing hub
    • Type: string
    • Default value: CATDlgEnable
  • ApplyChkFastPartRelWithoutFltr
    Role: Load From Manufacturing Hub - Filter features based on filters on related parts
    • Type: string
    • Default value: CATDlgDisable
  • ApplyIndexPartFastRel
    Role: Load From Manufacturing Hub - Load sort index for feature part relation
    • Type: string
    • Default value: CATDlgDisable
  • ApplyPartLoadConstraint
    Role: Fastener Assignment - Part loading constraint
    • Type: string
    • Default value: CATDlgDisable
  • ApplyRemoveWeldDuringSync
    Role: Load From Manufacturing Hub - Remove non-assigned fasteners during synchronization
    • Type: string
    • Default value: CATDlgDisable
  • ApplyTSAOption
    Role: Task Specification Activity - Delete while removing fasteners if nothing else is present
    • Type: string
    • Default value: CATDlgDisable
  • AssignmentStatusColor
    Role: Fasteners Assignment - Assignment status color
    • Type: string
    • Default value: CATDlgDisable
  • AttribsDefaultFileName
    Role: Read fastener attribute from text file - Default file name for the text document
