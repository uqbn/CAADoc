SectioningRepository Setting Repository
Quick Reference

SectioningRepository

Role: Settings related to the Section command

The SectioningRepository repository has the following characteristics:

  • Primary Tab Page: /Digital Mockup/DMU Space Analysis/DMU Sectioning
  • Repository File: SectioningRepository
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

SectioningRepository setting repository attributes:

  • GridAbsoluteMode
    Role: Section grid mode
    • Type: integer
    • Default value: 0
    • Default Value Role: Default grid mode is absolute
    • Values:
      0 Absolute
      1 Relative
  • GridAutoResize
    Role: Autoamtic grid resizing
    • Type: integer
    • Default value: 1
    • Values:
      0 Grid is not resized
      1 Grid is resized
  • GridLineStyle
    Role: Section grid style
    • Type: integer
    • Default value: 0
    • Default Value Role: Default grid style is Lines
    • Values:
      0 Lines
      1 Crosses
  • GridLodsDisplay
    Role: Autoamtic grid filtering
    • Type: integer
    • Default value: 0
    • Values:
      0 Grid is not filtered
      1 Grid is filtered
  • GridStepHaut
    Role: Section step height
    • Type: float
    • Default value: 100
    • Default Value Role: Default height is 1000 mm
  • GridStepLarg
    Role: Section step width
    • Type: float
    • Default value: 100
    • Default Value Role: Default width is 1000 mm
  • PlaneDefaultsColor
    Role: Default color of the section plane
    • Type: unsigned integer
    • Default value: 0
    • Default Value Role: Green in RGB order
    • Min value: 0
    • Max value: 255
  • PlaneInNoShow
    Role: Hide or show the section plane
    • Type: integer
    • Default value: 0
    • Values:
      0 The plane is hidden
      1 The plane is shown
  • PlaneNormale
    Role: Direction of section plane
    • Type: unsigned integer
    • Default value: 1
    • Default Value Role: Default direction of section plane is X
    • Values:
      1 X
      2 Y
      3 Z
  • PlaneOrigine
    Role: Origin of the section plane
    • Type: integer
    • Default value: 1
    • Default Value Role: Default origin is at the selection
    • Values:
      0 Origin
      1 Selection
  • ResultInNoShow
    Role: Hide or show the section result
    • Type: integer
    • Default value: 1
    • Values:
      0 The result is hidden
      1 The result is shown
  • SectionExportType
    Role: Section export
    • Type: integer
    • Default value: 1
    • Values:
      0 Use curves for export
      1 Use polyline for export
  • SectionFilled
    Role: Section Fill
    • Type: integer
    • Default value: 0
    • Values:
      0 Section appears filled
      1 Section does not appear filled
  • TileVertically
    Role: Result window
    • Type: integer
    • Default value: 0
    • Default Value Role: Default is window size
    • Values:
      0 WindowSize
      1 TileVertically
  • UpdateResult
    Role: Automatically compute the result
    • Type: integer
    • Default value: 0
    • Values:
      0 The result is computed
      1 The result is not computed
  • ViewerAutoOpen
    Role: Section viewer window display
    • Type: integer
    • Default value: 1
    • Values:
      0 No autoamtic open
      1 Automatic open
  • ViewerLock2D
    Role: Section in 2D viewer
    • Type: integer
    • Default value: 1
    • Values:
      0 No always 2D view
      1 Always 2D view
  • ViewerReframe
    Role: Section viewer reframe
    • Type: integer
    • Default value: 1
    • Values:
      0 Automatic reframe of the viewer
      1 No automatic reframe of the viewer
  • VolumeCutMode
    Role: Allow measure on section
    • Type: unsigned integer
    • Default value: 0
    • Values:
      0 Measure is not allowed
      1 Measure is allowed
  • WindowHeight
    Role: Result window height
    • Type: integer
    • Default value: 200
    • Default Value Role: Default height is 200
  • WindowWidth
    Role: Result window width
    • Type: integer
    • Default value: 200
    • Default Value Role: Default width is 200
  • Wireframe
    Role: Cut wireframe elements
    • Type: integer
    • Default value: 1
    • Values:
      0 Wireframe elements are not cut
      1 Wireframe elements are cut
