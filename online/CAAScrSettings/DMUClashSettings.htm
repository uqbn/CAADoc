DMUClashSettings Setting Repository
Quick Reference

DMUClashSettings

Role: Setting to manage the interference computation specifications

Detailed role:Setting to manage the interference computation specifications

The DMUClashSettings repository has the following characteristics:

  • Primary Tab Page: /Digital Mockup/DMU Space Analysis/DMU Clash
  • Repository File: Clash
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

DMUClashSettings setting repository attributes:

  • AutoComputeDepth
    Role: Mode to set the computation of results
    Detailed role: Mode to set the computation of results
    • Type: integer
    • Default value: 0
    • Default Value Role: By default, nothing is computed
    • Values:
      1 PenDepthMinDist
      2 AllResults
      0 None
  • ClashValuesFontSize
    Role: Font size for the clash values
    Detailed role: Font size for the clash values
    • Type: float
    • Default value: 3
  • LineAutoSelect
    Role: Whether first line is automatically selected
    Detailed role: Whether first line is automatically selected
    • Type: integer
    • Default value: 1
  • RetrieveInfo
    Role: Level Of Detail
    Detailed role: Level Of Detail
    • Type: integer
    • Default value: 0
    • Default Value Role: By default, do not retrieve information
    • Values:
      1 FromPreviousComputation
      2 FromPDM
      0 None
  • TabPageSelected
    Role: Mode of display of result
    Detailed role: Mode of display of result
    • Type: integer
    • Default value: 1
    • Default Value Role: By default, results are displayed by conflict
    • Values:
      1 Conflict
      2 Product
  • TypeOfComputation
    Role: Type of computation
    Detailed role: Type of computation
    • Type: integer
    • Default value: 2
    • Default Value Role: By default, the type of computation is Contact and Clash
    • Values:
      2 ContactClash
      3 ClearanceContactClash
  • ValueOfClearance
    Role: Value of clearance
    Detailed role: Value of clearance to be used in the computation
    • Type: double
    • Default value: 5
  • ViewerAutoOpen
    Role: Autoamtically open the viewer
    Detailed role: Autoamtically open the viewer
    • Type: integer
    • Default value: 0
