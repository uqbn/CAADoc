DMUDistance Setting Repository
Quick Reference

DMUDistance

Role: Setting to manage the XML export of Clash results

Detailed role:Setting to manage the XML export of Clash results

The DMUDistance repository has the following characteristics:

  • Primary Tab Page: /Digital Mockup/DMU Space Analysis/DMU Distance
  • Repository File: Distance
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

DMUDistance setting repository attributes:

  • PublishDefDir
    Role: Path of the directory where the distance results will be generated
    Detailed role: Path of the directory where the distance results will be generated
    • Type: string
    • Default value: ""
  • ViewerAutoOpen
    Role: Automatically open the viewer
    Detailed role: Automatically open the viewer
    • Type: integer
    • Default value: 1
  • XslOutput
    Role: Browser opened automatically
    Detailed role: Browser opened automatically
    • Type: integer
    • Default value: 0
  • XslTemplate
    Role: Path of style sheet for the XML dump
    Detailed role: Path of style sheet for the XML dump
    • Type: string
    • Default value: ""
