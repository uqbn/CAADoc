ConstraintsAndDimensions Setting Repository
Quick Reference

ConstraintsAndDimensions

Role: Setting to manage Constraints and Dimensions visualization options

Detailed role:Constraints and Dimensions setting contains all options necessary to manage Constraints and Dimensions visualization

The ConstraintsAndDimensions repository has the following characteristics:

  • Primary Tab Page: /General/Parameters and Measures/Constraints and Dimensions
  • Repository File: Constraint
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

ConstraintsAndDimensions setting repository attributes:

  • Angle
    Role: Attribute to define the constraint view angle
    Detailed role: Defines the angle in degrees from which the constraint is displayed in the geometry area, when you are rotating the geometry. This angle is measured between the normal to the screen and the constraint direction.
    • Type: double
    • Default value: 1
    • Default Value Role: By default the view angle is 1 deg
    • Min value: 0
    • Max value: 180
  • Dim3dFilterStatus
    Role: Constraint Filter panel
    Detailed role: Attribute to define combination of settings on Constraint Filter panel.
    • Type: integer
    • Default value: 4097
    • Default Value Role: By default, all 11 checkboxes are checked and grayed out
  • Display_Mode
    Role: Attribute to define the constraint display mode
    Detailed role: Defines the constraint display mode.
    • Type: integer
    • Default value: 0
    • Default Value Role: By default only the constraint value is displayed
    • Values:
      0 The constraint value is displayed
      1 The constraint name is displayed
      2 The constraint value and name are displayed
      3 The constraint valuen name and formula are displayed
  • Display_Param_Mode
    Role: Attribute to Display Mode available as Knowledge parameter
    Detailed role: Defines whether Display Mode available as Knowledge parameter.
    • Type: integer
    • Default value: 0
    • Default Value Role: By default Display Mode not available as Knowledge parameter
    • Values:
      0 Display Mode not available as Knowledge parameter
      1 Display Mode available as Knowledge parameter
  • Filter_ActivePartOnly
    Role: Constraint Filter panel
    Detailed role: Attribute to define combination of settings on Constraint Filter panel.
    • Type: integer
    • Default value: 4096
    • Default Value Role: By default, two check boxes under Product are un-checked.
  • Filter_Angle
    Role: Attribute to display angles constraints
    Detailed role: Defines whether angles constraints are displayed.
    • Type: integer
    • Default value: 1
    • Default Value Role: By default the angles constraints are displayed
    • Values:
      0 Constraints are not displayed
      1 Constraints are displayed
  • Filter_Chamfer
    Role: Attribute to display chamfer constraints
    Detailed role: Defines whether chamfer constraints are displayed.
    • Type: integer
    • Default value: 1
    • Default Value Role: By default the chamfer constraints are displayed
    • Values:
      0 Constraints are not displayed
      1 Constraints are displayed
  • Filter_Coincidence
    Role: Attribute to display coincidence constraints
    Detailed role: Defines whether coincidence constraints are displayed.
    • Type: integer
    • Default value: 1
    • Default Value Role: By default the coincidence constraints are displayed
    • Values:
      0 Constraints are not displayed
      1 Constraints are displayed
  • Filter_Concentric
    Role: Attribute to display concentricity constraints
    Detailed role: Defines whether concentricity constraints are displayed.
    • Type: integer
    • Default value: 1
    • Default Value Role: By default the concentricity constraints are displayed
    • Values:
      0 Constraints are not displayed
      1 Constraints are displayed
  • Filter_Constrained
    Role: Attribute to display driving constraints

    • Type: integer
    • Default value: 1
    • Default Value Role: By default the driving constraints are displayed
    • Values:
      0 Constraints are not displayed

  • Filter_Distance
    Role: Attribute to display dimension constraints

  • Gap
    Role: Attribute to define the distance between the constrained geometries and constraint extension lines
    Detailed role: Defines the distance between the constrained geometries and constraint extension lines, in mm.
    • Type: double
    • Default value: 2
    • Default Value Role: By default the view angle is 2 mm
  • Hilite_pointed_elements
    Role: Attribute to highlight the constrained geometries

      0 The constrained geometries are not highlighted
      1 The constrained geometries are highlighted

    Role: Attribute to define the distance between the constraint dimension lines and constraint extension lines
    Detailed role: Defines the distance between the constraint dimension lines and constraint extension lines, in mm.
    • Type: double
    • Default value: 2
    • Default Value Role: By default the view angle is 2 mm
  • Scale
    Role: Attribute to define the dimension style options
    Detailed role: Defines the size of the graphical symbols for iconic constraint.
    • Type: integer
    • Default value: 1
    • Default Value Role: By default medium symbols are displayed
    • Values:
      0 Displays small symbols
      1 Displays medium symbols
      2 Displays large symbols
  • Show_Color_Status
    Role: Attribute to display the status color
    Detailed role: Defines if the default status color or the graphical user defined color is displayed on constraint.
    • Type: integer
    • Default value: 0
    • Default Value Role: By default the default status color is used
    • Values:
      0 The graphical user defined color is used
      1 The Update command will be automatically launched
  • Show_iconic_line
    Role: Attribute to display the iconified constrained elements
    Detailed role: Defines whether the iconified constrained elements are displayed.
    • Type: integer
    • Default value: 1
    • Default Value Role: By default iconified constrained elements are displayed
    • Values:
      0 Does not display iconified constrained elements
      1 Displays iconified constrained elements
  • Status_geoinv
    Role: Invalid Geometry Constraints color in 3D viewer
    Detailed role: This attribute defines the color status for constraints set on a geometry became invalid in the 3D viewer. The three integer are the Red, Green, Blue coefficients
