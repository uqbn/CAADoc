DNBSimStatic Setting Repository
Quick Reference

DNBSimStatic

Role: Settings to handle the Process Verification Parameters

The DNBSimStatic repository has the following characteristics:

  • Primary Tab Page: /DPManufacturing/Verification
  • Repository File: DNBSimStatic
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

DNBSimStatic setting repository attributes:

  • DNBPVCenterActivity
    Role: Scrolls the PPRActivity list to the center
    Detailed role: Option when checked on Scrolls the PPR list to the centre during simulation
    • Alias: DNBPVCenterActivity
    • Type: integer
    • Default value: 0
  • DNBPVERSupport
    Role: Supports Engineering Requirements during PV
    Detailed role: This option when checked provides support for Engineering Requirements during PV
    • Alias: DNBPVERSupport
    • Type: integer
    • Default value: 0
  • DNBPVExpandTree
    Role: Expands the PPR Activity list at all levels
    Detailed role: This option when checked on expands the PPR activity list nodes at all levels. Is enabled only if the Scroll activity to centre option is enabled
    • Alias: DNBPVExpandTree
    • Type: integer
    • Default value: 0
  • DNBPVInheritHideShow
    Role: Inherits the Initial Hide show status of the products
    Detailed role: With this option checked on, the initial hide show status of the products is respected
    • Alias: DNBPVInheritHideShow
    • Type: integer
    • Default value: 0
  • DNBPVNavigationDown
    Role: Navigates to the First child when down button is pressed during PV
    Detailed role: With this option checked, on pressing the down button during PV, navigates down to the first child
    • Alias: DNBPVNavigationDown
    • Type: integer
    • Default value: 0
  • DNBPVPCPSupport
    Role: Supports the output relation Process Created Product
    Detailed role: This option when checked on provides support for the Process Creates Product output relation during PV
    • Alias: DNBPVPCPSupport
    • Type: integer
    • Default value: 0
  • DNBPVSuppressLinkWarnings
    Role: Suppresses link warnings during PV
    Detailed role: If the option is checked on, the link warning messages are suppresed during PV
    • Alias: DNBPVSuppressLinkWarnings
    • Type: integer
    • Default value: 0
