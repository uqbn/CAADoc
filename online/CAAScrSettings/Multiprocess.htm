Multiprocess Setting Repository
Quick Reference

Multiprocess

Role: SMultiprocess settings

The Multiprocess repository has the following characteristics:

  • Primary Tab Page: /Digital Mockup/Multiprocess Settings
  • Repository File: Multiprocess
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

Multiprocess setting repository attributes:

  • LogFilePath
    Role: Path of the log file
    • Type: URN
    • Default value: UNIX: ${CATTemp} Windows: ${CATTemp}
  • MaxSon
    Role: Maximum number of child processes
    • Type: integer
    • Default value: 2
    • Min value: 1
    • Max value: 100
    • Step: 1
  • MaxTime
    Role: Timeout in minutes
    • Type: integer
    • Default value: 60
    • Min value: 1
    • Max value: 720
    • Step: 1
