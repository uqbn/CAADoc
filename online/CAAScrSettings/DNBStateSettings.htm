DNBStateSettings Setting Repository
Quick Reference

DNBStateSettings

Role: Settings to handle the State Management Parameters

The DNBStateSettings repository has the following characteristics:

  • Primary Tab Page: /DPManufacturing/State Management
  • Repository File: DNBStateSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1
  • Setting controller:
    • Framework_CPP: DNBStateInterfaces
    • Framework_IDL: DNBStateInterfaces
    • Interface_CPP: DNBIDSIStateViewerSettingAtt
    • Interface_IDL: DNBIAStateViewerSettingAtt
    • LateType: DNBDSIStateViewerSettingCtrl

DNBStateSettings setting repository attributes:

  • CSPFilter
    Role: Displays Conditions, States and Positions in the hierarchical tree
    Detailed role: Displays Conditions, States and Positions in the hierarchical tree
    • Alias: CSPFilter
    • Type: integer
    • Default value: 0
