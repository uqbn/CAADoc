DNBGanttSettings

Role: Settings to handle the Gantt Chart Window Parameters

The DNBGanttSettings repository has the following characteristics:

  • Primary Tab Page: /DPManufacturing/Gantt Chart
  • Repository File: DNBGanttSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1
  • Setting controller:
    • Framework_CPP: DNBDpmInterfaces
    • Framework_IDL: DNBDpmInterfaces
    • Interface_CPP: DNBIDPMGanttSettingAtt
    • Interface_IDL: DNBIAGanttSettingAtt
    • LateType: DNBDPMGanttSettingCtrl
