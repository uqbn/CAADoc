What's New in V5R11 in CAA V5 Automation Objects

This article presents automation object, methods and properties that have been added since V5R11 GA to enhance CAA V5 automation capabilities, support functional evolutions and correct defects. Some of those modification already exist on V5R10 service packs. A few properties and methods have been modified or suppressed. VB and VBScripts code using those methods may have to be modified. VB projects using the corresponding objects must be recompiled. 

  • New Classes
  • New Methods and Properties
  • Modified Methods and Properties
  • Suppressed Classes, Methods and Properties

New articles or use cases in V5R11:

  • V5R11 GA
    • The single home page changes to accommodate more Solutions and Modelers. Point to the CATIA, DELMIA, or ENOVIA Portal brand on the top left corner of the home page, and select a solution to display its Automation home page. Then click the modeler(s) you want.
    • The following modelers are new:
      • 3D Functional Tolerancing & Annotation
      • Real Time Rendering
      • Product Engineering Optimizer
    • The Material Document Automation Objects
    • In Drafting
      • Creating a Drawing Table
      • Dimensions Checking
      • Duplicating a Drawing View
    • In Analysis
      • Preprocessing Commands
    • In Manufacturing
      • Insert Axial Tool Motion in Point To Point Activity
      • Insert Lathe Tool Motions in a Sequential Activity
