BatchInfrastructure Global Function CATBatchParamToSplitAccess


CATBatchParamToSplitAccess( iBatch,
oBatch,
oDescName)
Retrieves the XML parameter to be splitted in a splitter batch. Role: Retrieves the XML parameter to be splitted in a splitter batch This method takes as input the XML parameter file of the splitter batch. It retrieves the CATIBatchCAA of the XML parameter file to be splitted and its descriptor name.
Parameters:
iBatch
the interface representing the XML parameter file of the splitter batch (retrived by GetCATIBatchCAA() )
oBatch
the interface representing the XML parameter to split
Returns:
S_OK upon success
This object is included in the file: CATBatchAccessCAA.h
