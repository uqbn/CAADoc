Visualization Enumeration CATPickAttribut

enum CATPickAttribut {
  CATNoPickAttr,
  CATPickAttr
}
Mode type for the picking of graphic representation.
Values:
CATNoPickAttr
The representation is not pickable, the user cannot interact with it using the mouse
CATPickAttr
The repersentation is pickable, the user can interact with it
See also:
