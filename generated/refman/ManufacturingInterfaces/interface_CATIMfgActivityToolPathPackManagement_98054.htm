interface CATIMfgActivityToolPathPackManagement

Interface to implement on a manufacturing operation to authorize tool path pack/unpack management.

It gives access to Pack/Unpack interactive commands and the ability (through settings) to pack the tool path during computation.

This object is included in the file: CATIMfgActivityToolPathPackManagement.h
If needed, your Imakefile.mk should include the module: MfgItfEnv
