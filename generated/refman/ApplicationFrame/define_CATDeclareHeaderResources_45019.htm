ApplicationFrame Macro CATDeclareHeaderResources

Macro CATDeclareHeaderResources ()
Defines a method to get the header's resource file.
Role: Insert this macro in the header file (.h) of your header class and Use it in conjunction with CATImplementHeaderResources in the .cpp file.
This object is included in the file: CATCommandHeader.h
If needed, your Imakefile.mk should include the module: CATApplicationFrame
