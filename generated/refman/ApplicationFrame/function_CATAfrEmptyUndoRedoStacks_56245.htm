ApplicationFrame Global Function CATAfrEmptyUndoRedoStacks


CATAfrEmptyUndoRedoStacks()
Flushes the Undo and Redo stacks.
Role: This method flushes the Undo and Redo stacks.
The Undo/Redo mechanism is freezed until a new transaction is created.
Modifications done while the Undo/Redo mechanism is freezed will not be undoable.
Returns:
Returns always S_OK.
