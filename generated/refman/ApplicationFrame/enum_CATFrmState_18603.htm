ApplicationFrame Enumeration CATFrmState

enum CATFrmState {
  CATFrmUnavailable,
  CATFrmAvailable
}
Command Header Availability.
Values:
CATFrmUnavailable
The command is not accessible: the starter is disabled and its icon is greyed.
CATFrmAvailable
The command is accessible: the starter is available and its icon is normal.
This object is included in the file: CATCommandHeader.h
