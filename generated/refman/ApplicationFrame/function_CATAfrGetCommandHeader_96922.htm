ApplicationFrame Global Function CATAfrGetCommandHeader


CATAfrGetCommandHeader( iCommandHeaderId,
oCommandHeader)
Gets a pointer to a command header.
Role: This function looks for a CATCommandHeader identifier and returns a pointer to this command header.
Parameters:
iCommandHeaderId
The identifier of the CATCommandHeader.
oCommandHeader
The CATCommand.
Returns:
Returns S_OK on success. Returns E_FAIL if the identifier is not an existing CATCommandHeader identifier.
This object is included in the file: CATAfrCommandHeaderServices.h
