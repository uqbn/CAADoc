ApplicationFrame Class CATFrm3DCamera

Usage: you must use this class as is. You should never derive it.

public class CATFrm3DCamera


Deprecated:

Instead, use the factory CATCreateDefaultCamera in CATCameraToolkit to create a camera and the APIs of the interface CATICamera and CATI3DCamera
Implementation class of the CATI3DCamera interface.
Role:This implementation class provides methods to manage parameters of a 3D camera.
