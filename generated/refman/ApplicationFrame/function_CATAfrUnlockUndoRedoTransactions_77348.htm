ApplicationFrame Global Function CATAfrUnlockUndoRedoTransactions


CATAfrUnlockUndoRedoTransactions()
Unfreezes the Undo/Redo mechanism.
Role: This method cancels the action of CATAfrLockUndoRedoTransactions. New modifications done will be undoable as usual once a new transaction is created.
Returns:
Returns always S_OK.
This object is included in the file: CATAfrUndoRedoServices.h
