ApplicationFrame Global Function CATCreateDefaultCamera


CATCreateDefaultCamera( iCameraType,
opICamera)
Creates a camera with default parameters.
Parameters:
iCameraType
The type of camera.
opICamera
The address where the returned pointer to the interface CATICamera is located.
Returns:
S_OK
if the query succeeds
E_FAILED
if the camera creation fails.

CATCameraType
