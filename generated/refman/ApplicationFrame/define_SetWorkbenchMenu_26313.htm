ApplicationFrame Macro SetWorkbenchMenu

Macro SetWorkbenchMenu (variableName,menu)
Merges the workbench's menubar with the workshop's menubar.
Parameters:
variableName
The pointer to the workbench's instance. This pointer is created by
NewAccess with CATCmdWorkbench as classname parameter.
menu
The pointer to the menu's instance created by
NewAccess with CATCmdContainer as classname
This object is included in the file: CATCreateWorkshop.h
