ApplicationFrame Macro SetAddinMenu

Macro SetAddinMenu (variableName,menu)
Merges the addin's menubar with the workbench's menubar.
Parameters:
variableName
The pointer to the addin's instance. This pointer is created by
NewAccess with CATCmdContainer as classname parameter.
menu
The pointer to the menu's instance created by
NewAccess with CATCmdContainer as classname
