CATIAApplicationFrame Macro CAT_EDITOR_DECLARE_FACTORY

Macro CAT_EDITOR_DECLARE_FACTORY (MODULE,EDITOR,TYPE)
Declares an editor.
Role:macro used to implement an editor factory interface.
Parameters:
MODULE
module defining editor.
Legal values: NOTHING
EDITOR
This object, which derives from
CATEditor, is created by the factory.
TYPE
not used, has to be the same as editor.
See also:
CATIUserSettings, CATIEditProperties
This object is included in the file: CATEditorFactory.h
If needed, your Imakefile.mk should include the module: CATIAApplicationFrame
