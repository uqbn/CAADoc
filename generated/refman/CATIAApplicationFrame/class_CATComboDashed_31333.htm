CATIAApplicationFrame Class CATComboDashed

Usage: you must use this class as is. You should never derive it.

public class CATComboDashed

Class to create a combo to choose a line type.
Role:This class constructs a combo to select a line type among 7 types:

  1. Solid
  2. Dotted
  3. Dashed
  4. Dot-Dashed
  5. Phantom
  6. Small Dotted
  7. JIS Axis

When a line type is selected, a CATDlgComboSelectNotification notification is sent, and you can retrieves the index of the line type thanks to the GetSelect
See also:
CATComboColor, CATComboWeight, CATComboSymbol
