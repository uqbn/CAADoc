GMOperatorsInterfaces Enumeration CATFrFTopologicalSweepAutomaticCoupling_Type

enum CATFrFTopologicalSweepAutomaticCoupling_Type {
  CATFrFTopologicalSweepAutomaticCoupling_Standard,
  CATFrFTopologicalSweepAutomaticCoupling_D1,
  CATFrFTopologicalSweepAutomaticCoupling_D1AndThenD2,
  CATFrFTopologicalSweepAutomaticCoupling_VertexCoupling
}
Automatic coupling types.
Values:
CATFrFTopologicalSweepAutomaticCoupling_Standard
Standard coupling: by curvilinear length.
CATFrFTopologicalSweepAutomaticCoupling_D1
D1 coupling: in each domain, coupling of the vertices of G0 and not G1 continuity (each domain being limited by 2 guides if any).
CATFrFTopologicalSweepAutomaticCoupling_D1AndThenD2
D1 and then D2 coupling: after a D1 coupling, the D2 coupling is done (vertices of G1 and not G2 continuity) in each domain defined by the D1 coupling.
CATFrFTopologicalSweepAutomaticCoupling_VertexCoupling
Vertex coupling: coupling in each domain limited by 2 guides, the vertex number i of the first profile with the vertex number i of each profile.
