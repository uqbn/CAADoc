GMOperatorsInterfaces Global Function CATCGMCreateRecomposeShells


CATCGMCreateRecomposeShells( iFactory,
iData,
iBodyToRecompose)
Creates a CATICGMRecomposeShells operator.
Parameters:
iFactory
The pointer to the geometry factory.
iData
The pointer to the data defining the configuration and the journal. If the journal inside iData is NULL, it is not filled.
iBodyToRecompose
The pointer to the body to be recomposed.
Returns:
[out, IUnknown#Release] The pointer to the created operator. To be released with the Release method after use.
