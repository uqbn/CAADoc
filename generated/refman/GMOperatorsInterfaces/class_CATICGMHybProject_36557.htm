GMOperatorsInterfaces Class CATICGMHybProject

Usage: you must use this class as is. You should never derive it.

public class CATICGMHybProject

Class defining the operator that projects a body on another one.
It follows the general frame of all operator and satisfies to the smart mechanism: the input bodies are not modified. A new resulting body is created, possibly sharing data with the input bodies.

  • CATICGMHybProject is created with the CATCGMCreateTopProject global function. It must be directly released with the Release method after use. It is not streamable.
  • The intersection can be restricted on a given domain on each body with the methods SetFirstDomainToIntersect and SetSecondDomainToIntersect
  • The result is accessed with the GetResult method. If you do not want to keep the resulting body, use the CATICGMContainer.Remove method to remove it from the geometric factory.
