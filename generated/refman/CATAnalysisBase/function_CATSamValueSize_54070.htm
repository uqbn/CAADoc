CATAnalysisBase Global Function CATSamValueSize


CATSamValueSize( iValue,
oSize)
Returns the size in bytes needed to store a value of a specified type.
Parameters:
iValue
The type of value.
oSize
The size in bytes of a value of type iValue.
Returns:
S_OK upon success, E_FAIL upon failure.
See also:
CATSamValue
