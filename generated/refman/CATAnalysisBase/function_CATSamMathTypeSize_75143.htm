CATAnalysisBase Global Function CATSamMathTypeSize


CATSamMathTypeSize( iMathType,
iMathDimension,
oNbComponents)
Returns the number of components of a mathematical type for a particular dimension.
Parameters:
iMathType
The mathematical type.
iMathDimension
The mathematical dimension (from 1 to 6).
oNbComponents
The number of components of the mathematical type iMathType with dimension iMathDimension.
Returns:
S_OK upon success, E_FAIL upon failure.
See also:
CATSamMathType
