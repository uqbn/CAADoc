public GetReference()
Retrieves the component associated with the node.
Role: This method allows you to retrieve a pointer on the component displayed in an object tree. It is the same component which implements CATINavigateObject.
The CATINavigModify and the CATINavigElement interfaces are implemented on a late type which is not the component itself. Thanks to the CATINavigElement.GetAssociatedInstance method and the current one, you have a pointer on the component.
Returns:
A component pointer.
