MechanicalModelerUI Typedef CATImportAgentBehavior

typedef unsigned long CATImportAgentBehavior
CATFeatureImportAgent behaviors.
Role:In the CATFeatureImportAgent.CATFeatureImportAgent or in the CATFeatureImportAgent.SetAgentBehavior methods, a specific behavior can be set for the CATFeatureImportAgent selection agent.
Values:
CATFIAgDecodeResultNotKept
To deactivate the optimization mode which keeps the decode result
CATFIAgCheckTypesAfterImport
In assembly context, the imported selection is also checked with the interfaces of type set with the
CATPathElementAgent.AddElementType or the CATPathElementAgent.SetOrderedTypeList methods.
