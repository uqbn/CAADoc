MecModInterfaces Interface CATIMfLine

Usage: an implementation of this interface is supplied and you must use it as is. You should not reimplement it.

interface CATIMfLine

Interface to filter features whose associated topology is a line.
Role: Query this interface on a mechanical feature. If the query succeeds, the topology associated with the feature is a line.

This object is included in the file: CATIMfLine.h
If needed, your Imakefile.mk should include the module: CATMecModInterfaces
