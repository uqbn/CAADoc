MecModInterfaces Interface CATIMmiGeometricalSet

Usage: an implementation of this interface is supplied and you must use it as is. You should not reimplement it.

interface CATIMmiGeometricalSet

Interface identifying a surfacic body.
Role:This interface is specific for surfacic sets, in other words for Geometrical Set and Ordered Geometrical Set features. The Body feature does not implement this interface.

See also:
CATIMmiOrderedGeometricalSet, CATIMmiNonOrderedGeometricalSet

This object is included in the file: CATIMmiGeometricalSet.h
If needed, your Imakefile.mk should include the module: CATMecModInterfaces
