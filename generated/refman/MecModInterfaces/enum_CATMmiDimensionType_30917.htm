MecModInterfaces Enumeration CATMmiDimensionType

enum CATMmiDimensionType {
  CATMmiInternal,
  CATMmiBorder,
  CATMmiFree,
  CATMmiDimTypeUnknown
}
Types used to specify the geometry position of BRep feature.

Role: This enum used to specify options for the geometry position of BRep feature. ( CATIMmiUseMfBRep )
Values:
CATMmiInternal
The geometry pointed is internal. An edge between two faces for example.
CATMmiBorder
The geometry pointed is the border of another one. An edge bording one face only, for example.
CATMmiFree
The geometry pointed is isolated. An edge part of a wire, for example.
CATMmiDimTypeUnknown
Returned value for error/particular cases
