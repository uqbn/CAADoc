interface CATIMf3DAxisSystemManager

Interface to manage the axis systems.
Role:This interface enables you:

  • To know all the axis systems created by the CATIMf3DAxisSystemFactory interface into a Part document. See the GetAxisSystems method.
  • To set or retrieve the current axis system. See SetCurrentAxisSystem and GetCurrentAxisSystem respectively.
  • To translate the coordinates of a point ( ConvertPointCoordinates ) or a vector ( ConvertVectorCoordinates ) in the current axis system.

This interface is implemented by the specification container of the Part document. You retrieve it by at least two means:
  • CATIContainerOfDocument.GetSpecContainer
  • CATInit.GetRootContainer
See also:
CATIMf3DAxisSystem, CATIMf3DAxisSystemFactory

o SetCurrentAxisSystem
public virtual SetCurrentAxisSystem( const iAxisSystem)
Sets an axis system as the current one.
Role:This method enables you to set an axis system as current. It guarantees that only one axis system is current in the Part document. Set an axis system the current one can be useful:
  • To use the ConvertPointCoordinates and ConvertVectorCoordinates to translate the point/vector coordinates in this axis system.
  • In an interactive session, to control the current axis system which is used by some commands such as Point. These commands can display points (or vectors ) coordinates in the current axis system.
Parameters:
iAxisSystem
The new current axis system. If the value is NULL_var, there is no current axis system, the coordinates are always displayed in the absolute axis system.
Returns:
E_FAIL if the iAxisSystem is not orthogonal - CATAxisSystemIsDirect
