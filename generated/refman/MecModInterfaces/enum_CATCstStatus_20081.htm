MecModInterfaces Enumeration CATCstStatus

enum CATCstStatus {
  CstStatus_ERROR,
  CstStatus_True,
  CstStatus_False,
  CstStatus_False_Imp_Attr,
  CstStatus_False_Val_Attr,
  CstStatus_False_Value,
  CstStatus_False_Usr,
  CstStatus_False_Broken,
  CstStatus_False_VisuMode
}
Status of a constraint.
Values:
CstStatus_True
Valid.
CstStatus_False
Invalid.
CstStatus_False_Imp_Attr
Invalid because specifications.
CstStatus_False_Val_Attr
Invalid because specifications.
CstStatus_False_Value
Invalid because value.
CstStatus_False_Usr
Invalid because user.
CstStatus_False_Broken
Invalid because missing geometry.
CstStatus_False_VisuMode
Invalid because geometry cannot be accessed (Visualization Mode).
