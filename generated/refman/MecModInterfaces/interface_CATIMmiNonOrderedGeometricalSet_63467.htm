MecModInterfaces Interface CATIMmiNonOrderedGeometricalSet

Usage: an implementation of this interface is supplied and you must use it as is. You should not reimplement it.

interface CATIMmiNonOrderedGeometricalSet

Interface identifying a Geometrical Set.
Role:This interface is specific for the Geometrical Set. The Ordered Geometrical Set does not implement this interface. It implements the CATIMmiOrderedGeometricalSet interface.
Ordered Geometrical Set and Geometrical Set are both surfacic sets, in other words GSMTool features (see CATIGSMTool ).
