MecModInterfaces Interface CATIMmiOrderedGeometricalSet

Usage: an implementation of this interface is supplied and you must use it as is. You should not reimplement it.

interface CATIMmiOrderedGeometricalSet

Interface identifying an Ordered Geometrical Set.
Role:This interface is specific for the Ordered Geometrical Set. The Geometrical Set does not implement this interface. It implements the CATIMmiNonOrderedGeometricalSet interface.
Ordered Geometrical Set and Geometrical Set are both surfacic sets, in other words GSMTool features (see CATIGSMTool ).
