SketcherInterfaces Interface CATI2DOffsetCurve

Usage: an implementation of this interface is supplied and you must use it as is. You should not reimplement it.

interface CATI2DOffsetCurve

Interface to filter offset curves.
Role: Query this interface on a sketched geometry. If the query succeeds, the feature is an offset curve which reference geometry is a sketched parametric curve that belongs to the sketch.

This object is included in the file: CATI2DOffsetCurve.h
If needed, your Imakefile.mk should include the module: CATSketcherInterfaces
