ObjectModelerBase Global Function CATIsDocLocatorActive


CATIsDocLocatorActive( const iLocatorType,
oResult)
Determines if a document locator is active.
Parameters:
iLocatorType
The locator's late type.
oResult
TRUE - the locator is active.
FALSE - the locator is not active.
