TopologicalOperatorsLight Global Function CATCreateTopVertex


CATCreateTopVertex( iFactory,
iData,
iPoint)
Creates an operator to build a point body from one point.
Parameters:
iFactory
The pointer to the factory of the geometry.
iData
The pointer to the data defining the software configuration and the journal. If the journal inside iData is NULL, it is not filled.
iPoint
The pointer to the point used to define the vertex.
Returns:
The pointer to the created operator. To delete with the usual C++ delete operator.
This object is included in the file: CATTopVertex.h
If needed, your Imakefile.mk should include the module: CATTopologicalOperatorsLight
