CATIAV4Interfaces Global Function CATV4iGisels


CATV4iGisels( const iSet,
const iPrevElement,
oNextElement,
oEnd,
ier)
Scans the elements of a given set of the current workspace.
Parameters:
iSet
[in] The set to scan
iPrevElement
[in] The previous element
Legal values: NULL to get the first element of the set
oNextElement
[out] The following element.
oEnd
[out] The end of the scan.
Legal values: value is 1 if iPrevElement is the last element
ier
error code
