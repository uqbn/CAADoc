CATIAV4Interfaces Global Function CATV4iGmlesc


CATV4iGmlesc( const iElement,
const iIdPrevious,
oIdLink,
oEnd,
ier)
Scans the ultra-weak links referencing an entity.
Parameters:
iElement
[in] The V4 element supporting the link(s)
iIdPrevious
[in] The identifier of the previous link
Legal values: 0 to start the scan
oIdLink
[out] The identifier of the link following iIdPrevious
oEnd
[out] The validity of oIdLink :
Legal values:
  • 1 if iIdPrevious is the last link
  • 0 if oIdLink is the link following iIdPrevious
ier
[out] error code
