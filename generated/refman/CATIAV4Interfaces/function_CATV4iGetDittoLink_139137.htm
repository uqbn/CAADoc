CATIAV4Interfaces Global Function CATV4iGetDittoLink


CATV4iGetDittoLink( const iElement,
oElement,
ier)
Retrieves the ditto element linked to the given connector occurrence.
Parameters:
iElement
[in] The pointer to the V4 tubing connector occurrence.
oElement
[out] The pointer to the V4 tubing ditto element linked to the connector occurrence.
ier
[out] The error code.
Returns:
  • 0 if succeeded
  • -1 otherwise
