CATIAV4Interfaces Global Function CATV4iGirtex


CATV4iGirtex( const iElement,
const iNumText,
oNbc,
oTxt,
oPt,
oSize,
oDir,
ier)
Retrieves a space text linked to an element.
Parameters:
iElement
[in] The V4 space element
iNumText
[in] The number of the text to be read
oNbc
[out] number of used characters.
Legal values: 0 if the text is the element's identifier
oTxt
[out] The text
oPt
[out] The coordinates of the text's positioning point
oSize
[out] The text size
Legal values: between 1 and 4
oDir
[out] text direction
Legal values:
  • 0: horizontal
  • 1: vertical
ier
[out] error code
