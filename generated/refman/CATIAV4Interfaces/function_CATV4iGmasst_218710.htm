CATIAV4Interfaces Global Function CATV4iGmasst


CATV4iGmasst( const iSetType,
const iPrevSet,
oNextSet,
oEnd,
ier)
Scans the application sets of a given type in a workspace.
Parameters:
iSetType
[in] The type of the application set
iPrevSet
[in] The previous application set
Legal values: NULL to start the scan
oNextSet
[out] The next application element of thr desired type
oEnd
[out] end of scan.
Legal values: 1 if iPrevSet is the last set
ier
[out] error code
