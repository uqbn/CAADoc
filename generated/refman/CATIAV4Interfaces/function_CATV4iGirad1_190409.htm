Retrieves the element called iIdent in the document.
Parameters:
iDoc
[in] The document to analyse. It must be a .model
iType
[in] The catgeo type of the element to search. If =0, search in all the geometrical element.
iCarNbr
[in] The length of the string iIdent
iIdent
[in] The identifier of the searched element
oElement
[out] The searched element
ier
[out] error code
