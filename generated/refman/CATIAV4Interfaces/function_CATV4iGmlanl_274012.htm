CATIAV4Interfaces Global Function CATV4iGmlanl


CATV4iGmlanl( const iElement,
const iIdLink,
oUserInfo,
oRefElement,
oUpdated,
ier)
Analyzes an ultra-weak link.
Parameters:
iElement
[in] The V4 element supporting the link(s)
iIdLink
[in] The identifier of the link
oUserInfo
[out] Data associated to the link
oRefElement
[out] The referenced element
oUpdated
[out] The synchronization between the link and the referenced element
Legal values:
  • 0 if it is synchronized
  • 1 if the referenced element has been modified after the last update of the link
ier
[out] error code
