interface CATINavigateObjectPPR

Interface to navigate in object tree.
Role: This interface enables you to visualize a component in the PPR tree. This interface implementation is mandatory,even if the component is a leaf.

This object is included in the file: CATINavigateObjectPPR.h
If needed, your Imakefile.mk should include the module: CATProcessInterfaces
