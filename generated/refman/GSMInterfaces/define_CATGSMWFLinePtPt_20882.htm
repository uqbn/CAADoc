GSMInterfaces Macro CATGSMWFLinePtPt

Macro CATGSMWFLinePtPt ()
Interface to line feature.
Role: Allows you to set/change the type of the line and to modify the orientation of the line. Line types :

Legal values : LinePtPt = 0 LinePtDir = 1 LineAngle = 2 LineTangency = 3 LineNormal = 4 LineBitangent = 5 LineBisecting = 6 LineExplicit = 30
This object is included in the file: CATIGSMLine.h
