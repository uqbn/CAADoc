PSNInteroperability Global Function CATCommitVPM


CATCommitVPM()
Commit in VPM.
Role:This method commits the pending transaction in the ENOVIAvpm V4 database.
This method must be called after CATSaveInVPMFromCATIA or CATSaveInVPM.

If needed, your Imakefile.mk should include the module: CATPinServices

Returns:
  • S_OK: Commit successful
  • E_FAIL: Commit failed
This object is included in the file: CATBatchServices.h
