Drafting2DLInterfaces Enumeration CAT2DLViewSide

enum CAT2DLViewSide {
  View2DLTopSide,
  View2DLBottomSide,
  View2DLLeftSide,
  View2DLRightSide,
  View2DLTLCorner,
  View2DLTRCorner,
  View2DLBLCorner,
  View2DLBRCorner
}
CAT2DLViewSide is used in conjonction with a reference View2DL and a ViewBox.

Role:It then determines precisely a View2DL type in the ViewBox. example: with the default ViewBox: - Front + TopSide = Top. - Right + LeftSide = Front. - Front + TRCorner = isometric on the corner near Front, Right, Top.
This object is included in the file: CAT2DLViewSide.h
