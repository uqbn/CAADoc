DraftingInterfaces Enumeration CATDrwUnitDisplayMode

enum CATDrwUnitDisplayMode {
  CATDrwUnitDisplayNone,
  CATDrwUnitDisplayAll
}
The display mode of units in a text (if the text contains (a) parameter(s) with unit(s)).
Values:
CATDrwUnitDisplayNone
Do not display any unit.
CATDrwUnitDisplayAll
Display all units.
