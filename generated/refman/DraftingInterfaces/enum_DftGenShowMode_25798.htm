DraftingInterfaces Enumeration DftGenShowMode

enum DftGenShowMode {
  HIDDEN,
  VISIBLE,
  OVERLOADED,
  NOT_DISPLAYED
}
Visualisation mode of the generated geometry.
The geometry can be :
Values:
HIDDEN
Edge of a part hidden behind an other one.
VISIBLE
Standard mode.
OVERLOADED
Some of the edges can have their graphic properties predefined.
NOT_DISPLAYED
For example the edge joining the two side surfaces on a cylinder.
This object is included in the file: CATIDftGenGeom.h
