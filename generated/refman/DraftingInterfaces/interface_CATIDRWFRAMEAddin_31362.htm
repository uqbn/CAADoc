Usage: you can freely reimplement this interface.

interface CATIDRWFRAMEAddin

Interface to add toolbars in the Drafting workbench.
by implementing this interface you create an Addin of the working view workshop of the drawing document. Its means that the toolbars and the item in the menu will be always there.

See also:
CATIWorkbenchAddin

This object is included in the file: CATIDRWFRAMEAddin.h
If needed, your Imakefile.mk should include the module: CATDraftingInterfaces
