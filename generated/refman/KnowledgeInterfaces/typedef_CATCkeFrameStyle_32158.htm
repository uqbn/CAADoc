KnowledgeInterfaces Typedef CATCkeFrameStyle

typedef unsigned long CATCkeFrameStyle
Possible values for defining panel ParamFrame.

Legal values: CATCkeLabel default style: with a label. CATCkeNoLabel no label CATCkeNoFormula if you do not want formulas in contextual menu. CATCkeWithSpinner if you wish a spinner instead of an editor CATCkeNoTolerance if you do not want tolerance CATCkeNoRange if you do not want ranges CATCkeNoCxtMenu if you do not want contextual menus at all CATCkeNoCtxMenuEdit if you do not want the "Edit..." contextual menu CATCkeNoCxtMenuMultipleValues if you do not want the "Multiple Values" contextual menu CATCkeNoJournaling if you do not want journaling of parameter modification CATCkeRightAttachment if you want a right attachement CATCkeNoMeasure if you don't want the Measure contextual menu CATCkeDTConfigFirstOutParameter when editing a design table configuration, shows the first associated parameter CATCkeRelationButtonDisabled if the parameter has a relation, the button appears but is disabled CATCkeEditMultipleForVolatile if a contextual menu for editing multiple values for volatile parameters is required CATCkeKeepCxtMenuWhenDisable if you want to keep the contextual menu even when the ParamFrame is disable CATCkeWithUnsetMenu if you want,in contextual menu, the capability to unset the parameter value.
