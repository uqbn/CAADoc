GMModelInterfaces Class CATCGMTessTrianIter

Usage: you must use this class as is. You should never derive it.

public class CATCGMTessTrianIter

Class defining the iterator on the triangles resulting from a Tessellation operator.

Such iterator is directly created by the CATICGMBodyTessellator.GetFace or CATICGMCellTessellator.GetFace methods. Its deletion is managed at the CATICGMBodyTessellator and CATICGMCellTessellator deletion.
Use it according to the following scheme:

  • Retrieve the number of triangles
  • Go the next result
  • Retrieve the result
  • Optionally, reset to begin again.

This class also allows you to directly retrieve the results in an array that you have to allocate and deallocate after use.
