GMModelInterfaces Global Function CATCGMCreatePlanarMapping


CATCGMCreatePlanarMapping( iFactory,
const iAxis)
Creates an object defining a global mapping between a plane and a 2D space.
The mapping corresponds to a projection onto a plane. This object is used by the CATICGMTopDevelop operator to define the input and output mappings. You have to Release this object after use.
Parameters:
iFactory
The pointer to the factory of the geometry.
iAxis
The axis defining the plane.
Returns:
[out, IUnknown#Release] The pointer to the created operator. To be released with the Release method after use.
This object is included in the file: CATCGMCreateSurTo2DCanonicalMapping.h
