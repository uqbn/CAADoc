VPMInterfaces Macro ENOVRegisterEventData

Macro ENOVRegisterEventData (name)
Defines an event stream into the package.

Role: Carries extra information about raised events.
This stream can then be referenced in methods declarations.
Precondition: a package must have been previously declared (with ENOVRegisterPackage.P macro).
Click here to see a complete example.

Parameters:
name
The event stream name.
