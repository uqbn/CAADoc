interface ENOVIUEVolumeComputing

User Exit to manage the items used to calculate object volume.

Role: Interface to be implemented in order to define the participation of the item inside the volume calculation.

Returns:

Legal values:

S_FALSE
The item does not participate in the volume calculation.
FAILED
An ERROR occured.
