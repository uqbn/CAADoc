VPMInterfaces Macro ENOVRegisterEventArg

Macro ENOVRegisterEventArg (stream,name,type)
Defines an event stream argument.
Parameters:
stream
The event stream name, defined with
ENOVRegisterEventData macro.
name
The event stream argument name.
type
The event stream argument type. See
ENOVEvtArgType for valid types.
This object is included in the file: ENOVRegister.h
