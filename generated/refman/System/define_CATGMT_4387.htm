System Macro CATGMT

Macro CATGMT ()
Defines a possible value for the argument iFlag of the method CATTime::ConvertToString.
If CATGMT is not specified in the iFlag argument, ConvertToString will return a string representation of the local time & date corresponding to the CATTime instance. If CATGMT is specified in the argument iFlag of the ConvertToString method, the generated string will be a string representation of the GMT time & date (instead of the local time).
This object is included in the file: CATTime.h
