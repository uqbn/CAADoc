System Enumeration CATSubscribeEndTransaction

enum CATSubscribeEndTransaction {
  1,
  CATSubscribeIdle
}
Command subscribing type.
Role: The subscribing type , used as parameter of the methods Subscribe and RemoveSubscribe.
Values:
CATSubscribeEndTransaction
The subscribing command must be called when the transaction is completed
CATSubscribeIdle
The subscribing command must be called when the process enters the main event loop
See also:
CATApplication.Subscribe, CATApplication.RemoveSubscribe
