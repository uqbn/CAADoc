System Global Function CATFFlush


CATFFlush( iFileDesc)
Flushes a file.
Role:Writes any buffered data for the previously opened with CATFOpen file and leaves it open
Parameters:
iFileDesc
the file descriptor that has been obtained through
CATFOpen
Returns:
Legal values:
S_OK :on Success.
negative : on failure
This object is included in the file: CATStdIO.h
