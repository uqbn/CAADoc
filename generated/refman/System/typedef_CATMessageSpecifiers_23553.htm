System Typedef CATMessageSpecifiers

typedef unsigned int CATMessageSpecifiers
Specifier for a backbone message class.

Role: This specifier is used in the CATICommMsg.GetMessageSpecifiers and in the CATICommMsg.SetMessageSpecifiers.
Values:
CATMsgSpec_AnswerNeeded
The sender requests an answer
CATMsgSpec_FirstReceiverOnly
The message will be sent to the first valid destination application only
CATMsgSpec_Reflexive
The sender requests to receive its own message (this is rarely needed)
This object is included in the file: CATMessageClass.h
