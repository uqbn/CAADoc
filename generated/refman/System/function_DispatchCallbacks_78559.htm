System Global Function DispatchCallbacks


DispatchCallbacks( iEventPublisher,
iPublishedNotification)
Dispatches the callbacks set to a given event publisher for a given notification.
Role: This function calls all the callback methods declared by all the subscribers that subscribe to the event publisher for the notification.
Parameters:
iEventPublisher
The event publisher
iPublishedNotification
The published notification
This object is included in the file: CATEventSubscriber.h
If needed, your Imakefile.mk should include the module: JS0GROUP
