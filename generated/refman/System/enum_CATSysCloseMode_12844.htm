System Enumeration CATSysCloseMode

enum CATSysCloseMode {
  CATSys_Close,
  CATSys_DeleteOnClose
}
The possible actions to take when closing documents.
Values:
CATSys_Close
Closes the document.
CATSys_DeleteOnClose
Deletes the document on close.
