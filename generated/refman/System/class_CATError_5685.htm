System Exception CATError

Usage: you can derive this class.

public class CATError

Base class for errors.
Role: You cannot instantiate this class. Derive it to create your own error class that refines the type of managed errors using CATDeclareError and CATImplementError macros.

o GetNLSSource
public virtual GetNLSSource()
Returns the error source.
Role: The error source is made of the concatenation of:
  • the source context. It is the name of the application that issues the error, usually CATIA
  • the error message catalog file name
  • the error message key in the error message catalog
They are separated with the "-" (dash) character.
Example: CATIA - SystemError - KeyRangeERR_3000
