DialogEngine Notification CATStateCancelNotification

Usage: you must use this class as is. You should never derive it.

public class CATStateCancelNotification

Class representing a notification whose instance is received when a state command is cancelled.
Role: As a command developer you don't need to create an instance of this class.

Cancel notification type.

The command completes normally because it has reached the NULL state.
Interrupt
The command ends because it is cancelled by an exclusive command.
