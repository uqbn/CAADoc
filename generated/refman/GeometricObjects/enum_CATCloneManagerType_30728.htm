GeometricObjects Enumeration CATCloneManagerType

enum CATCloneManagerType {
  CatCGMSingleDuplicate,
  CatCGMFullDuplicate,
  CatCGMUnknownDuplicate
}
Defines the mode of the clone process.
Values:
CatCGMSingleDuplicate
Only duplicates the objects that are Added to the CATCloneManager
CatCGMFullDuplicate
Duplicates the objects that are Added to the CATCloneManager and the forward linked objects.
CatCGMUnknownDuplicate
Internal use.
This object is included in the file: CATCloneManager.h
