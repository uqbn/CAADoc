interface CATIPrdHandleImpactsOnReference

Interface to rename a Publication and manage the impacts.

Role : By using this interface for renaming a publication, impacted elements pointing this publication will be reconnected.

Warnings :

Only pointing objects loaded in the session will be reconnected.

This interface is implemented on ASMProduct References only.

Implementation is only avalaible under PX1 license.

o RenamePublication
public virtual RenamePublication( const iOldName,
const iNewName)
Renames a Publication and reconnect its pointing elements.
Role: This method renames the publication and changes automatically the paths of pointing objects loaded in session.
Parameters:
iOldName
Name of the existing publication, needed to retrieve it
iNewName
The new name of the publication
Returns:
The possible return values are :
  • E_FAIL: the new name is not set on publication successfully. Publication has not been found, or its new name is not valid :
    • forbidden char : "!", ":", "<", ">"
    • not valuated
    • starts with an empty char : " "
  • S_OK: the new name is set on publication successfully and all pointing elements are reconnected.
