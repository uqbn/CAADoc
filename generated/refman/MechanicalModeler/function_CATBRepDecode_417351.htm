MechanicalModeler Global Function CATBRepDecode


CATBRepDecode( const iCell,
const iAssociatedFeature,
const iDecodeType,
const iSelectType= MfSimpleSelect)
Returns the selecting object from a topological cell and the topological body of a specified feature.
Parameters:
iCell
Topological cell.
iAssociatedFeature
Feature owner of the topological body used to select the topological cell.
iDecodeType
Legal values: MfDecodeAll, MfDecodeWire, MfDecodeShell, MfDecodeLump.
iSelectType
Legal values: MfForceSelect, MfSimpleSelect.
