VisualizationBase Enumeration CATPickPriority

enum CATPickPriority {
  PICK_DEFAULT,
  PICK_WIREFRAME
}
Picking priority.

Role: Enum used to specify the type of sort to use in the CATPickPathList.SortGraphicAttributeSet method.
Values:
PICK_DEFAULT
Deep is used first, then dimension.
PICK_WIREFRAME
Dimension is used first, then deep.
This object is included in the file: CATPickPathList.h
