VisualizationBase Enumeration CATCameraType

enum CATCameraType {
  CAT3DConicCamera,
  CAT3DCylindricCamera,
  CAT3DUnknownCamera,
  CAT2DCamera
}
Camera type.
Values:
CAT3DConicCamera
for a 3D perspective projection
CAT3DCylindricCamera
for a 3D Cylindric projection
CAT3DUnknownCamera
for any 3D type projection
CAT2DCamera
for a 2D view
