VisualizationBase Enumeration CATSecondaryPixelImageFormat

enum CATSecondaryPixelImageFormat {
  sRGB,
  sRGBA,
  sABGR,
  sARGB
}
Format of the secondary pixel data stored in a CATPixelImage.
Values:
sRGB
Red, Green, Blue - 24 bits images.
sRGBA
Red, Green, Blue with Alpha - 32 bits.
sABGR
Alpha, Blue, Green, Red - 32 bits images.
sARGB
Alpha, Red, Green, Blue - 32 bits.

CATPixelImage
