VisualizationBase Enumeration CATVizCGRAccessMode

enum CATVizCGRAccessMode {
  CATVizCGRAccessModeDirect,
  CATVizCGRAccessModeMemory
}
Defines the method used for writing the CGR.

Role:This context is used in CATReadCgr and CATWriteCgr .
Values:
CATVizCGRAccessModeDirect
The cgr file will be streamed directly to disk (slower, but no size limitations
CATVizCGRAccessModeMemory
The cgr file will be streamed in memory and then to the disk
This object is included in the file: CATVizGlobals.h
