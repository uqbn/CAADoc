VisualizationBase Global Function ClearLods


ClearLods( ldata,
sdata,
size)
Function to free useless memory.
Role: This function is to be used after a call to MakeLods, in order to free useless memory in the LodData <***output> and <**sag> arguments of MakeLods.
Parameters:
ldata
Array of LodData parameters to be cleared (same format as MakeLods).
sdata
Array of sags allocated by the previous routine.
size
Number of LodData to be cleared.
