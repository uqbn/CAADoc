VisualizationBase Enumeration CATReadOnlyState

enum CATReadOnlyState {
  ReadOnly,
  NotReadOnly
}
Defines whether a CATPixelImage can be modified or not.
Values:
ReadOnly
No modifications allowed.
NotReadOnly
The image can be modified.
