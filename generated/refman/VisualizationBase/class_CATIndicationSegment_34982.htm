VisualizationBase Class CATIndicationSegment

Usage: you must use this class as is. You should never derive it.

public class CATIndicationSegment

Class to define the line segment under the mouse cursor in a 3D viewpoint.
Role: This class stores in its public data members pt1 and pt2 the extremities of the line segment located under the mouse cursor. This class is used by dialog objects when making a query to a manipulator.
