The style of the push button. Legal values:

Creates a button which will trigger a specific event when maintained pressed.
CATDlgPusNoShadow
Button without 3D look. This style is effective only if an icon is defined for the button.
CATDlgBtnIconAndText
When they are defined, both its icon and its title will be displayed on the button face. This style is ignored if CATDlgPusNoShadow is specified.
