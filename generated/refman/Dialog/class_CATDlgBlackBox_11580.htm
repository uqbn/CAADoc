Dialog Class CATDlgBlackBox

Usage: you must use this class as is. You should never derive it.

public class CATDlgBlackBox

Base class for CATDlgWindows (for Windows platforms) or CATDlgMotif (for Unix platforms).
You cannot construct a CATDlgWindows directly: it can be constructed as the base class of one of its derived class only.

This object is included in the file: CATDlgBlackBox.h
If needed, your Imakefile.mk should include the module: DI0PANV2
