AdvancedTopologicalOpe Global Function CATCreateTopSpine


CATCreateTopSpine( iGeoFactory,
iTopData,
iProfiles)
Creates a CATTopSpine operator.
Parameters:
iFactory
The pointer to the factory of the geometry.
iTopData
The pointer to topological data for operator configuration.
iProfiles
The list of pointers to profiles (CATBody) or planes (CATPlane).
This object is included in the file: CATTopSpine.h
If needed, your Imakefile.mk should include the module: CATAdvancedTopologicalOpe
