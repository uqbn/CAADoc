CATAnalysisInterfaces Enumeration CATSamDataType

enum CATSamDataType {
  CATSamDataTypeSet,
  CATSamDataTypeEntity,
  CATSamDataTypeCharac,
  CATSamDataTypeList,
  CATSamDataTypeMath,
  CATSamDataTypeApply,
  CATSamDataTypeLast
}
The types of objects which are present in the field model.
Values:
CATSamDataTypeSet
A set.
CATSamDataTypeEntity
An entity.
CATSamDataTypeCharac
A simple characteristic.
CATSamDataTypeList
A list.
CATSamDataTypeMath
Internal use only.
CATSamDataTypeApply
Internal use only.
CATSamDataTypeLast
No object type defined.
It also represents the number of valid enumerated values.
This object is included in the file: CATAnalysisExplicitDefinitions.h
