CATAnalysisInterfaces Enumeration SAM_Status

enum SAM_Status {
  Status_Invalid,
  Status_Valid,
  Status_ValidPre,
  Status_Computed,
  Status_Restart,
  Status_Singular
}
Analysis Set status.
Values:
Status_Invalid
The status is not valid. Update is required.
Status_Valid
The status is valid according to the specifications.
Status_ValidPre
The status is valid according to the Finite Element view.
Status_Computed
The status is valid according to the solver computation.
Status_Restart
The status is valid according to the solver computation. Capability to restart the computation.
Status_Singular
The status is valid according to the solver computation. Some trouble are detected (like local singularity).
