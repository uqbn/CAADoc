CATAnalysisInterfaces Enumeration SAM_Quality

enum SAM_Quality {
  Sam_IN,
  Sam_OUT,
  Sam_NEUTRAL
}
Quality of a set inside a case regarding to the update capability.

Values:
Sam_IN
To define an input of a Case.
Sam_OUT
To define an output of a Case.
Sam_NEUTRAL
To define an Neutral behavoir. No impact in the update.
