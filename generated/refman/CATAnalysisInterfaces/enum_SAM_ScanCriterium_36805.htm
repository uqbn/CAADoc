CATAnalysisInterfaces Enumeration SAM_ScanCriterium

enum SAM_ScanCriterium {
  Scan_with_name,
  Scan_with_type,
  Scan_both
}
SAM_ScanCriterium: Enumerate to manage search varaibles.

Values:
Scan_with_name
The name of the set is a criteria for the search.
Scan_with_type
The type of the set is a criteria for the search.
Scan_both
The name and type of the set is a criteria for the search.
