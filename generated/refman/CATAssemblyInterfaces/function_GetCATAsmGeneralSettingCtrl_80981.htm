CATAssemblyInterfaces Global Function GetCATAsmGeneralSettingCtrl


GetCATAsmGeneralSettingCtrl( const iInterfaceIdentifier,
oInterfacePointer)
Returns the Assembly general settings controller.
Parameters:
iInterfaceIdentifier
The identifier of the interface which is queryied on the controller.
oInterfacePointer
The address where the returned pointer to the interface is located.
Returns:
S_OK
if the controller is successfully returned
E_NOINTERFACE
if the interface does not exist
Other
if the controller cannot be returned.
This object is included in the file: CATAsmGeneralSettingServices.h
