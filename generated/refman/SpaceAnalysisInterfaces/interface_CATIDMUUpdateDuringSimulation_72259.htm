interface CATIDMUUpdateDuringSimulation

This interface facilitates use of "Light Transaction Mechanism" to minimise memory consumption during update of DMU data.

This interface can be queried on a document. User should call BeginLightUpdateTransaction & EndLightUpdateTransaction at start & end of transaction.
