SpaceAnalysisInterfaces Typedef CATDMUDataType

typedef unsigned int CATDMUDataType
DMU Data Type to copy.
It's a DMU data type designed to convey which DMU data to copy. A combination of several data types is possible. For example: CATDMUSectionData | CATDMUMeasureData.
This object is included in the file: CATDMUCopyData.h
