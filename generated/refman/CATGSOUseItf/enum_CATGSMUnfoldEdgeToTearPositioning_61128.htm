CATGSOUseItf Enumeration CATGSMUnfoldEdgeToTearPositioning

enum CATGSMUnfoldEdgeToTearPositioning {
  CATGSMUnfoldEdgeToTearUndefinedOrientation,
  CATGSMUnfoldEdgeToTearDefaultOrientation,
  CATGSMUnfoldEdgeToTearInversedOrientation
}
Define the edge to tear positioning orientation Role: Used when the reference origin is located on an edge to tear
Values:
CATGSMUnfoldEdgeToTearUndefinedOrientation
The orientation is undefined
CATGSMUnfoldEdgeToTearDefaultOrientation
The orientation is the default one
CATGSMUnfoldEdgeToTearInversedOrientation
The orientation is inversed
