o LogImplicitAxis
public static LogImplicitAxis( const piAxisSet,
const iAxis)
Logs the creation of an Explicit Axis. This feature must be used in order log the creation of an CATAnalysisExplicitAxis when this creation is done during the explicitation of an boundary condition, load or property. The axis life cyle is manage by the feature that manage the explicitation (By including the entity in the NewExplicitlist of CATISamExplicitation after calling the UpdateExplicitList). But to be collected under the axis feature set, the axis must be also given to this Global Method.
Parameters:
piAxisSet
the AxisSet feature.
iAxis
the explicit axis created.

This explicit axis must be created with the explicit object associated to piAxisSet as parent.
