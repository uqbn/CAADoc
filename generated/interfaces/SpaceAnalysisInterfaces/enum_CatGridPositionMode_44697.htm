CatGridPositionMode (Enumeration)

enum CatGridPositionMode {
  catGridPositionMode_Absolute,
  catGridPositionMode_Relative
}
The different modes for section grid positioning.
It is used by the SectioningSettingAtt object to specify the section grid's positioning mode in settings.
Values:
catGridPositionMode_Absolute
The grid's positioning is absolute.
catGridPositionMode_Relative
The grid's positioning is relative.
