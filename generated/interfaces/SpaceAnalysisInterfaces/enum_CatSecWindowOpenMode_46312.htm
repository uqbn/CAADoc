CatSecWindowOpenMode (Enumeration)

enum CatSecWindowOpenMode {
  catSecWindow_DefaultSize,
  catSecWindow_TileVertically
}
The different modes of opening the sectioning windows It is used by the
SectioningSettingAtt object to specify the mode of opening the section window in settings.
Values:
catSecWindow_DefaultSize
Open the sectioning window with the default size specified in the Tools->Options.
catSecWindow_TileVertically
Tile the sectioning window vertically in the viewer
