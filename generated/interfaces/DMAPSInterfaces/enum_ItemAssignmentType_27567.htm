ItemAssignmentType (Enumeration)

enum ItemAssignmentType {
  ProcessProcesses,
  ProcessFirstProcesses,
  ProcessRemoves,
  ProcessImplementsRequirementPartial,
  ProcessImplementsRequirementComplete,
  productviewpoint,
  productvisualization
}
The Analysis Level setting attribute range of values.
Values:
ProcessProcesses
Assignment type is Process Processes Item/Product
ProcessFirstProcesses
Assignment type is Process First Processes Item/Product
ProcessRemoves
Assignment type is Process Removes Item/Product
ProcessImplementsRequirementPartial
Assignment type is Process Implements Requirement(Partial)with Item/Product
ProcessImplementsRequirementComplete
Assignment type is Process Implements Requirement(Complete)with Item/Product
