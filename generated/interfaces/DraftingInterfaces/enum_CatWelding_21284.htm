CatWelding (Enumeration)

enum CatWelding {
  catNoneWelding,
  catFirstWelding,
  catSecondWelding
}
Active drawing welding.
Role: CatWelding is used to specify which welding is active, that is the one you can work on.
Values:
catNoneWelding
No welding is set as active
catFirstWelding
The first welding is set as active
catSecondWelding
The second welding is set as active
