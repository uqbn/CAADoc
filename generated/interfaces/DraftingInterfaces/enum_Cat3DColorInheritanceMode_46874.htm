Cat3DColorInheritanceMode (Enumeration)

enum Cat3DColorInheritanceMode {
  cat3DColorInheritanceModeOff,
  cat3DColorInheritanceModeOn
}
View color mode.
Values:
Cat3DColorInheritanceModeOff
The view is drawn with hidden lines seen
Cat3DColorInheritanceModeOn
The view inherits the colors of 3D elements
