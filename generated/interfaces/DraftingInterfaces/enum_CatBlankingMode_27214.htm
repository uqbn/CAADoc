CatBlankingMode (Enumeration)

enum CatBlankingMode {
  catBlankingInactive,
  catBlankingActive,
  catBlankingOnGeom
}
Annotation blanking mode.
Role: Defines the mirror mode of the text.
Values:
catBlankingInactive
No blanking : blanking is not active
catBlankingActive
blanking is active on area fill only
catBlankingOnGeom
blanking is active on area fill AND on geometry
