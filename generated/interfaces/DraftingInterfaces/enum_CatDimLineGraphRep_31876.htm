CatDimLineGraphRep (Enumeration)

enum CatDimLineGraphRep {
  catDimLine1Part,
  catDimLine2Parts,
  catDimLineLeader1Part,
  catDimLineLeader2Part
}
Dimension line graphical representation.
Role: CatDimLineGraphRep is used to specify dimension line graphical representation.
Values:
catDimLine1Part
Dimension line one part.
catDimLine2Parts,
Dimension line two parts.
catDimLineLeader1Part,
Dimension line with one part leader.
catDimLineLeader2Part,
Dimension line with two part leader.
