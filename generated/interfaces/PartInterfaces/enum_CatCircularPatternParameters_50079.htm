CatCircularPatternParameters (Enumeration)

enum CatCircularPatternParameters {
  catInstancesandAngularSpacing,
  catCompleteCrown,
  catUnequalAngularSpacing
}
Circular Pattern Parameters.
The Circular Pattern can apply parameters to the angular repartition.
Values:
catInstancesandAngularSpacing
The number of instances and the angular spacing are used to define the Pattern.
catCompleteCrown
Only the number of instances is used to define the Pattern, the total angle is set to 360 degrees.
