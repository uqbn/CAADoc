CatChamferOrientation (Enumeration)

enum CatChamferOrientation {
  catNoReverseChamfer,
  catReverseChamfer
}
Chamfer orientation.
Defines whether to keep the face selected or determined by CATIA as the chamfer reference face.
Values:
catNoReverseChamfer
The chamfer reference face is the face selected or determined by CATIA
catReverseChamfer
The chamfer reference face is the other face
