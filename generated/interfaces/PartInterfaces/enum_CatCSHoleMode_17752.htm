CatCSHoleMode (Enumeration)

enum CatCSHoleMode {
  catCSModeDepthAngle,
  catCSModeDepthDiameter,
  catCSModeAngleDiameter
}
Countersunk hole mode.
It describes the chosen mode for countersunk hole.
Values:
catCSModeDepthAngle
Countersunk parameters are depth and angle
catCSModeDepthDiameter
Countersunk parameters are depth and Diameter
catCSModeAngleDiameter
Countersunk parameters are angle and Diameter
