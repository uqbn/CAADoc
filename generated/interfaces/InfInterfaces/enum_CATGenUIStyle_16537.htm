CATGenUIStyle (Enumeration)

enum CATGenUIStyle {
  UIStyleP1,
  UIStyleP2,
  UIStyleP3
}
The User Interface Style setting attribute range of values.
Role: This enum is used in the GeneralSessionSettingAtt interface.
Values:
UIStyleP1
Look is P1
UIStyleP2
Look is P2
UIStyleP3
