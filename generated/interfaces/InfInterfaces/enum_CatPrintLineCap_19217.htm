CatPrintLineCap (Enumeration)

enum CatPrintLineCap {
  catPrintFlat,
  catPrintSquare,
  catPrintRound
}
CatPrintLineCap.
It used by the PageSetup object for document printing.
Values:
catPrintFlat
The line ends are printed flat.
catPrintSquare
The line ends are printed squared.
catPrintRound
The line ends are printed round.
