CatTreeSizeTypeEnum (Enumeration)

enum CatTreeSizeTypeEnum {
  catTreeSizeTypeFixedSize,
  catTreeSizeTypeTextDependentSize
}
Types for the size of the text in the specification tree.
Role: This enum is used in the TreeVizManipSettingAtt interface.
Values:
catTreeSizeTypeFixedSize
Only a fixed number of characters are shown in the specification tree, for names of its items.
catTreeSizeTypeTextDependentSize
The integrality of each item's name is shown.
