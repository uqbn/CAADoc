SystemConfiguration (Object)

Provides abstractions to resources which depend on the platform or the current configuration.

Property Index

OperatingSystem
Returns a string which identifies the operating system on which the application is currently running.
ProductCount
Returns the number of product names names currently known to the system.
Release
Returns the CATIA release number.
ServicePack
Returns the CATIA service pack number.
Version
Returns the CATIA version number (usually version 5).

Method Index

GetProductNames
Returns the product names of all the licenses currently known to the system.
IsProductAuthorized
Returns True if the specified product is authorized, False otherwise.

Methods


o Sub GetProductNames( ioProductNames)
Returns the product names of all the licenses currently known to the system.
Parameters:
CATSafeArrayVariant
An properly dimensioned array of strings in which the product names will be stored.
Example:
This example determines if the first product in the list of known product names is authorized. 
 Dim SystemConfiguration1 As SystemConfiguration
 Set SystemConfiguration1 = CATIA.SystemConfiguration
 ReDim NameArray(SystemConfiguration1.ProductNamesCount)
 SystemConfiguration1.GetProductNames NameArray
 MsgBox "IsProductAuthorized for product " & NameArray(0) & "  returns " & SystemConfiguration1.IsProductAuthorized(NameArray(0))
o Func IsProductAuthorized( iProductName) As
Returns True if the specified product is authorized, False otherwise.
Parameters:
iProductName
The name of the product to check.
oAuthorized
A boolean which specifies if the product is authorized.
