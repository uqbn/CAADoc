CATSearchContextScope (Enumeration)

enum CATSearchContextScope {
  Everywhere,
  InWorkbench,
  FromWorkbench,
  FromSelection,
  VisibleOnScreen
}
The Default Power Input Context Scope setting attribute range of values.
Role: This enum is used in the SearchSettingAtt interface.
Values:
Everywhere
shortcut "all"
InWorkbench
shortcut "in"
FromWorkbench
shortcut "from"
FromSelection
shortcut "sel"
VisibleOnScreen
shortcut "scr"
