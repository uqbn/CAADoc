CatShowResultType (Enumeration)

enum CatShowResultType {
  ByRule,
  ByObject,
  ByState
}
Sorting option for the Expert Knowledge text report.
Values:
ByRule
Sort by the name of the rules.
ByObject
Sort by the name of the objects.
ByState
Sort by the status of the rules (failed or succeeded).
