CatWorkModeType (Enumeration)

enum CatWorkModeType {
  DEFAULT_MODE,
  VISUALIZATION_MODE,
  DESIGN_MODE
}
Possible types of Workmode.
Values:
DESIGN
The representation is in Design Mode
VISUALIZATION
The representation is in Visualization mode
DEFAULT
The representation is in Default Work mode.Default mode is a mode where the Product Container is loaded. It's an intermediate mode between Visualization Mode and Design Mode.
