CatPartUpdateMode (Enumeration)

enum CatPartUpdateMode {
  catManualUpdate,
  catAutomaticUpdate
}
Update mode setting parameter's possible values.
Role: This enum is used in the PartInfrastructureSettingAtt interface.
Values:
catManualUpdate
User has to manually launch update tasks.
catAutomaticUpdate
Update is automatically launched
