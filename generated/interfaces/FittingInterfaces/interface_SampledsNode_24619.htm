SampledsNode (Object)

The interface to access a CATIASampledsNode.
Role: The SampledsNode interface can be used to obtain different types of Sampled based collections from within the current action document.

Method Index

GetSampleds
Retrieves a sampleds based collection of objects

Methods


o Func GetSampleds( iCollectionName) As
Retrieves a sampleds based collection of objects
Parameters:
iCollectionName
The name of the collection to retrieve.
Returns:
The requested collection
Example:
The following example retrieves the collection of Track objects (called CATIATracks) from within the current document. 
 Dim myTracks as Tracks
 Set myTracks = SampledsNode.GetSampleds ("Tracks")
