CatStrLinkMode (Enumeration)

enum CatStrLinkMode {
  catStrWithLinkMode,
  catStrNoLinkMode,
  catStrRefRefWithLinkMode,
  catStrRefRefNoLinkMode
}
Associative mode.
Creating a cutback uses, in a member, contextual geometry from other parts. This geometry can be copied with or without a link on the original geometry.
Values:
catStrWithLinkMode
a links is kept
catStrNoLinkMode
no links is kept
catStrWithRefLinkMode
a reference links is kept
